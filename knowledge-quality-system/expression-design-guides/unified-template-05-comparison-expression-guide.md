# unified-template-05-comparison 表現設計完全ガイド

## 🎯 テンプレート表現特性
- **適用場面**: 複数選択肢の比較検討、メリット・デメリットの対比、判断基準の明確化
- **表現効果**: 客観的な判断支援、選択の根拠提示、比較による理解促進
- **セクション構成哲学**: 比較表形式により選択肢の特徴を明確化し、読者の意思決定を支援
