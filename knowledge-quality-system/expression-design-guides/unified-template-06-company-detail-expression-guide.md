# unified-template-06-company-detail 表現設計完全ガイド

## 🎯 テンプレート表現特性
- **適用場面**: 企業や組織の詳細情報提示、具体的な事例やケーススタディの説明
- **表現効果**: 深い理解の促進、具体性による説得力向上、詳細情報による判断支援
- **セクション構成哲学**: 詳細な情報提示により読者の理解を深化し、具体的な判断材料を提供
