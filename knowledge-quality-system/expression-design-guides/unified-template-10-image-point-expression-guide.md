# unified-template-10-image-point 表現設計完全ガイド

## 🎯 テンプレート表現特性
- **適用場面**: 画像と重要ポイントの組み合わせによる視覚的説明、キャラクターと要点の効果的な連携
- **表現効果**: 視覚的記憶の強化、親しみやすさの演出、重要ポイントの印象深い伝達
- **セクション構成哲学**: 画像の感情的効果とポイントの論理的説明を組み合わせ、理解と記憶を最大化
