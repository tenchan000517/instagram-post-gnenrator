# unified-template-03-ranking-display 表現設計完全ガイド

## 🎯 テンプレート表現特性
- **適用場面**: 順位付けや優先度付けで情報を整理、複数選択肢の序列化
- **表現効果**: 判断基準の明確化、選択の簡略化、重要度の可視化
