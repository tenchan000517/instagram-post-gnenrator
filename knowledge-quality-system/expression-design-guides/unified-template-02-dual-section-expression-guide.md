# unified-template-02-dual-section 表現設計完全ガイド

## 🎯 テンプレート表現特性
- **適用場面**: 対比構造や前後比較で理解を深める、2つの視点やアプローチを並列提示
- **表現効果**: 明確な構造による理解促進、選択肢の可視化、比較による判断支援
- **セクション構成哲学**: 2つのセクションの対比や補完により、情報の立体的な理解を促進
