## 📋 管理方針

### ✅ 格納すべきコンテンツ
- 投稿用に完成したコンテンツ
- Instagram投稿として使用するもの
- 最終成果物・完成品

### ❌ 格納すべきでないもの
- 作業用ファイル・下書き
- システムファイル・設定ファイル
- データベース・参考資料
