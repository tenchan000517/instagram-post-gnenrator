# 企業データベース活用フィードポスト生成システム

## 🎯 システム概要
既存の企業情報データベースシステム（5Phase構築フロー + Type003生成プロセス）をフィードポスト生成用に完全統合。companyMasterData.json + rankingGenerator.jsを活用した企業分析型コンテンツ生成を実現。

**対応データベース**:
- 企業データ: `../../data-systems/03_DATABASE/company-databases/companyMasterData.json`
- 拡張データ: `../../data-systems/03_DATABASE/company-databases/companyMasterData_extended.json`
- 生成エンジン: `../../data-systems/01_ACTIVE/ranking-engine/rankingGenerator.js`

**専門キャラクター**: Type003(KIKUYO) - データ分析・ランキング特化
