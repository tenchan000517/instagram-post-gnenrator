# アイデア発散プロンプト集

## 概要
Instagram投稿コンテンツの創造的アイデア発散システム。50技術統合アイデア発散、テーマ別ランキング拡張マトリクス、Dynamic Few-Shot Learning例示により、独創性と価値を両立したコンテンツアイデアを生成する。

---

## 50技術統合アイデア発散手法

### 基盤技術セット（10技術）

#### 1. Chain-of-Thought (CoT) 推論
```
【活用方法】
- 問題→分析→解決策の論理的思考展開
- ステップバイステップの思考プロセス可視化
- 複雑な概念の段階的理解促進

【Instagram適用例】
- Type002 KINGの実践ガイド（8ステップ展開）
- 問題解決プロセスの丁寧な説明
- 論理的思考力向上コンテンツ
```

#### 2. Few-Shot Learning 例示学習
```
【活用方法】
- 成功事例・失敗事例の対比提示
- パターン認識による理解促進
- 具体例から抽象概念への昇華

【Instagram適用例】
- 「成功者の共通習慣 vs 失敗パターン」
- 「良い例・悪い例の比較」
- 「Before/After変化事例」
```

#### 3. Zero-Shot Reasoning ゼロショット推論
```
【活用方法】
- 事前知識なしでの論理的推論
- 原理原則からの演繹的思考
- 普遍的法則の発見・適用

【Instagram適用例】
- 基本原理からの応用展開
- 「なぜそうなるのか」の根本理解
- 業界を超えた普遍的ノウハウ
```

#### 4. Self-Consistency 自己一貫性
```
【活用方法】
- 複数角度からの検証・確認
- 矛盾のない論理構築
- 信頼性の高い結論導出

【Instagram適用例】
- 3つの視点からの分析（個人・組織・社会）
- 専門家意見の統合・整理
- 複数データソースでの裏付け
```

#### 5. Meta-Prompting メタプロンプティング
```
【活用方法】
- AI自身によるプロンプト最適化
- 思考プロセスの改善・進化
- より効果的な問いかけの生成

【Instagram適用例】
- 「より良い質問の作り方」
- 「思考の質を上げる方法」
- 「問題設定の重要性」
```

### 発展技術セット（20技術）

#### 6-10. 創造性向上技術群
```
【Analogical Reasoning】類推・比喩思考
- 異分野からのアイデア転用
- メタファーを使った理解促進

【Counterfactual Thinking】反実仮想思考  
- 「もし〜だったら」の仮想シナリオ
- 対照的な状況での効果確認

【Perspective Taking】視点転換
- 異なる立場からの考察
- 多角的な問題理解

【Constraint Satisfaction】制約充足
- 限られた条件での最適解発見
- 創造性を促す制約設定

【Divergent Thinking】拡散的思考
- 一つの問題から多数のアイデア創出
- 既成概念にとらわれない発想
```

#### 11-15. 論理強化技術群
```
【Deductive Reasoning】演繹推論
- 一般原則から具体的結論導出
- 確実性の高い論理展開

【Inductive Reasoning】帰納推論
- 個別事例から一般法則発見
- パターン認識による予測

【Abductive Reasoning】仮説推論
- 最も妥当な説明の仮説生成
- 限られた情報からの推論

【Causal Reasoning】因果推論
- 原因と結果の関係性分析
- 根本原因の特定・解決

【Temporal Reasoning】時系列推論
- 時間的変化・発展の理解
- トレンド予測・将来展望
```

#### 16-25. 実践最適化技術群
```
【Iterative Refinement】反復改善
- 段階的な品質向上プロセス
- フィードバックによる最適化

【Multi-Agent Collaboration】マルチエージェント協調
- 複数の専門家視点統合
- 集合知による問題解決

【Socratic Questioning】ソクラテス式問答
- 深掘り質問による真理探求
- 前提の見直し・再検討

【Design Thinking】デザイン思考
- ユーザー中心の問題解決
- 創造的かつ実用的なソリューション

【Systems Thinking】システム思考
- 全体最適化の視点
- 相互関係・循環構造の理解
```

## Meta-Prompting（AI自己最適化）

### 自己改善ループシステム

#### Level 1: 基本メタプロンプト
```
【メタプロンプト実行指示】
あなたは今、Instagram投稿のアイデアを生成しようとしています。
より良いアイデアを生成するために、まず以下を考えてください：

1. 【現在の問いかけの改善】
   - この問いかけで本当に良いアイデアが出るか？
   - より具体的・効果的な問い方はないか？
   - 追加すべき条件・制約はないか？

2. 【思考プロセスの最適化】
   - どの思考技術を組み合わせるべきか？
   - どの順序で考えるのが最適か？
   - 見落としがちな視点はないか？

3. 【出力品質の向上】
   - どのような形式が最も使いやすいか？
   - 追加すべき情報・データは何か？
   - 実用性を高める要素は何か？

改善された問いかけと思考プロセスで、再度アイデア生成を実行してください。
```

#### Level 2: 高度メタプロンプト
```
【高度メタプロンプト実行指示】
あなたは50の技術を統合したアイデア生成の専門家です。
以下の手順でメタ認知的にアイデア生成プロセスを最適化してください：

1. 【コンテキスト分析】(30秒)
   - ターゲット層の詳細分析
   - 競合コンテンツの差別化ポイント
   - 現在のトレンド・社会情勢の考慮

2. 【技術選択】(60秒)
   - 50技術から最適な5-7技術を選択
   - 技術の組み合わせによる相乗効果検討
   - 各技術の適用順序・重み付け決定

3. 【プロンプト自動生成】(90秒)
   - 選択した技術に基づく最適プロンプト生成
   - 制約条件・評価基準の明確化
   - 期待する出力形式・品質基準の設定

4. 【実行・評価・改善】(180秒)
   - 生成されたプロンプトでアイデア創出
   - 出力品質の自己評価（10項目×10点）
   - 不足要素の特定・プロンプト改善

最終的に、最高品質のアイデアセット（3-5個）を提示してください。
```

### プロンプト進化システム

## Reflexion（自己評価・改善ループ）

### 3段階自己評価システム

#### Stage 1: 即時評価
```
【生成直後の自己チェック】
1. アイデアの新規性（1-10点）
   - 既存コンテンツとの差別化度
   - 独創性・ユニーク性

2. 実践可能性（1-10点）
   - ユーザーの実行しやすさ
   - 必要リソース・コスト

3. 価値提供度（1-10点）
   - ターゲットへの有用性
   - 問題解決・改善効果

4. Instagram適合性（1-10点）
   - ビジュアル化しやすさ  
   - シェア・保存される可能性
```

#### Stage 2: 深層分析
```
【30分後の振り返り評価】
1. 論理的一貫性確認
   - 主張と根拠の整合性
   - ストーリーの流れ・構成

2. 感情的訴求力評価
   - 共感・感動を呼ぶ要素
   - 行動変容を促す動機付け

3. 競合優位性分析
   - 既存コンテンツとの差異
   - 独自価値・強み

4. リスク・制約事項検討
   - 実装時の障壁・課題
   - 予想される批判・反論
```

#### Stage 3: 長期的視点評価
```
【24時間後の総合評価】
1. 持続性・発展性
   - 長期的な価値・影響
   - 関連コンテンツへの展開可能性

2. ブランド適合性
   - 全体戦略・方向性との整合
   - キャラクター・トーンとの合致

3. 社会的影響・責任
   - 社会に与える影響・価値
   - 倫理的問題・配慮事項

4. 戦略的重要性  
   - ビジネス目標への貢献度
   - 長期的なROI・価値創造
```
