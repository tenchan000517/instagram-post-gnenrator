## 2-3: 5軸評価システム（TEN品質基準）

【必須5軸評価】
1. **普遍性** (0-100): どれだけ多くの人・状況で活用可能か
2. **実用性** (0-100): 実際の業務・生活でどれだけ使えるか  
3. **実証性** (0-100): 効果がデータ・事例で証明されているか
4. **視覚性** (0-100): 効果・プロセスが分かりやすく可視化できるか
5. **簡潔性** (0-100): 学習・導入がシンプルで迅速か

【TEN総合スコア算出】
```
TENスコア = (普遍性×0.25) + (実用性×0.3) + (実証性×0.25) + (視覚性×0.1) + (簡潔性×0.1)
```
