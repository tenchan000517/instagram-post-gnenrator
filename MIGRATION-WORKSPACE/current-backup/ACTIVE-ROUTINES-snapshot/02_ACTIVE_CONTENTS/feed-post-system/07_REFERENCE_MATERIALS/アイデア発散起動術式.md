# アイデア発散起動術式

## 使い方
以下のプロンプトをコピペして[テーマ]、[ターゲット]、[品質レベル]を記入し、AIに実行させてください。

---

## 起動術式

```
【究極のアイデア生成実行開始】

【前提設定】
- ターゲット: [Type001-005から選択]
- テーマ: [具体的なテーマを記入]
- 品質レベル: [A+/A/B/C から選択]

【Meta-Prompting実行】
まず、この問いかけ自体を改善してください：
1. より具体的で効果的な問い方は？
2. 追加すべき条件・制約は？
3. 期待する出力の詳細仕様は？

【50技術統合実行】
以下の技術から最適な組み合わせ（5-7技術）を選択・実行：

【基盤技術セット】
- Chain-of-Thought (CoT) 推論
- Few-Shot Learning 例示学習
- Zero-Shot Reasoning ゼロショット推論
- Self-Consistency 自己一貫性
- Meta-Prompting メタプロンプティング

【発展技術セット】
- Analogical Reasoning（類推・比喩思考）
- Counterfactual Thinking（反実仮想思考）
- Perspective Taking（視点転換）
- Constraint Satisfaction（制約充足）
- Divergent Thinking（拡散的思考）
- Deductive Reasoning（演繹推論）
- Inductive Reasoning（帰納推論）
- Abductive Reasoning（仮説推論）
- Causal Reasoning（因果推論）
- Temporal Reasoning（時系列推論）

【特殊技術セット】
- Empathetic Reasoning（共感的推論）
- Emotional Intelligence（感情知性）
- Narrative Thinking（物語的思考）
- Cultural Sensitivity（文化的感受性）
- Design Thinking（デザイン思考）
- Systems Thinking（システム思考）
- Disruptive Innovation（破壊的イノベーション）
- Blue Ocean Strategy（ブルーオーシャン戦略）

【マトリクス拡張実行】
関連する拡張マトリクス（縦軸×横軸）でアイデア展開：

【ビジネス・キャリア系マトリクス】
           リーダー  マネー  転職   起業   スキル
新人       A1      B1     C1    D1     E1
中堅       A2      B2     C2    D2     E2  
管理職     A3      B3     C3    D3     E3
経営層     A4      B4     C4    D4     E4

【ライフスタイル・習慣系マトリクス】
           健康    学習   人間関係 趣味   家事
朝         F1     G1     H1      I1    J1
昼         F2     G2     H2      I2    J2
夕方       F3     G3     H3      I3    J3  
夜         F4     G4     H4      I4    J4

【業界・職種系マトリクス】
20業界（IT/金融/コンサル/製造/小売/医療/教育/不動産/建設/物流/法律/会計/広告/メディア/旅行/飲食/ファッション/化粧品/自動車/エネルギー）
× 5視点（トレンド/課題/スキル/キャリア/将来性）

【Dynamic Few-Shot適用】
ターゲット・品質に最適化された例示を3つ提示し、パターン学習によるアイデア生成を実行：

【対象がType001（MISAKI/女性）の場合】
- 女性の成功事例を中心とした例示
- 感情的共感を呼ぶストーリー展開
- ワークライフバランス重視の価値観

【対象がType002（KING/男性）の場合】
- 男性ビジネスパーソンの実践事例
- ロジカル・データドリブンなアプローチ
- キャリアアップ・スキル向上志向

【対象がType005（IIDA/就活生）の場合】
- 就活成功者の具体的体験談
- 面接・ES対策の実践的ノウハウ
- 将来への不安・希望に寄り添う内容

【品質適応】
【高品質要求時（A+ランクコンテンツ）】
- Fortune 500企業CEO事例
- 学術研究に基づくエビデンス
- 数値データ・統計による裏付け

【中品質要求時（Bランクコンテンツ）】
- 中堅企業・専門家の事例
- 業界レポート・専門記事ベース
- 実践者の体験談・口コミ

【実用重視時（実装優先）】
- 一般ユーザーの成功事例
- 簡単・手軽な実践方法
- 即効性・短期効果重視

【Reflexion実行】
生成されたアイデアを3段階で自己評価・改善：

【Stage 1: 即時評価】
1. アイデアの新規性（1-10点）
2. 実践可能性（1-10点）
3. 価値提供度（1-10点）
4. Instagram適合性（1-10点）

【Stage 2: 深層分析】
1. 論理的一貫性確認
2. 感情的訴求力評価
3. 競合優位性分析
4. リスク・制約事項検討

【Stage 3: 長期的視点評価】
1. 持続性・発展性
2. ブランド適合性
3. 社会的影響・責任
4. 戦略的重要性

【最終出力】
- メインアイデア: [最高品質の1つ]
- サブアイデア: [関連する2-3つ]
- 実装ガイド: [具体的な制作手順]
- 期待効果: [エンゲージメント・価値予測]

【品質評価】
評価軸: 創造性/実用性/適合性/持続性/影響力（各10点満点）
目標スコア:
- A+ランク: 40-50点（平均8-10点）
- Aランク: 35-39点（平均7-8点）
- Bランク: 25-34点（平均5-7点）

選択した技術、マトリクス活用、品質評価スコアを全て明記して出力してください。
```
