# 🚀 Instagram投稿 新規作成フローマスター

## 📋 新規作成フロー（6ステップ）

### **①リサーチプロンプトと起動術式**
**目的**: 元ネタ・調査結果・思考などのネタの種を収集・整理
- トレンド調査
- ユーザーニーズ分析
- 競合分析
- アイデア発散
- 情報ソース収集

### **②投稿タイプ別起動術式**
**目的**: 収集したネタを基に適切なTypeを判定・選択
- **Type001起動**: MISAKI - 女性の悩み共感系
- **Type002起動**: KING - 男性のキャリア実践系
- **Type003起動**: KIKUYO - ランキング・データ系
- **Type004起動**: TEN - 効率化・ツール系
- **Type005起動**: IIDA - 就活・学生系

### **③コンテンツ作成マスターガイドライン**
**目的**: Type別の品質基準に従ってコンテンツ作成
- Type別チェックリスト適用
- 感情フロー設計（Type001）
- 実践ステップ設計（Type002）
- データ構造設計（Type003）
- 実証内容設計（Type004）
- 就活ニーズ設計（Type005）

### **④テンプレート配置マスターガイドライン**
**目的**: 作成したコンテンツを最適なテンプレートに配置
- template-placement-ultimate-master.md
- Type別推奨テンプレート選択
- ページ構成最適化
- 情報階層設計

### **⑤キャラクター別フィード投稿用キャラクターシンクロドキュメント**
**目的**: キャラクターの個性・トーンを完全同期
- **MISAKI**: 共感的・温かい・寄り添い型
- **KING**: 実践的・論理的・成長志向型
- **KIKUYO**: 分析的・客観的・情報提供型
- **TEN**: 効率的・革新的・問題解決型
- **IIDA**: 応援的・親身・未来志向型

### **⑥キャプション・ハッシュタグ作成マニュアル**
**目的**: エンゲージメント最大化のための文言作成
- Type別キャプションテンプレート
- ターゲット別ハッシュタグ戦略
- CTA（Call to Action）設計
- 保存促進フレーズ

```
[①リサーチ・ネタ収集]
         ↓

         ↓
    [Type別分岐]

         ↓

         ↓
    [完成・出力]
```

- リサーチからの価値創造
- Type判定からの最適化
