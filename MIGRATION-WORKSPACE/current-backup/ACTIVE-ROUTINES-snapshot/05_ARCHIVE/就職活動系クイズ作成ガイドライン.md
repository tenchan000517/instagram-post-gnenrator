# 就職活動系クイズ作成ガイドライン

## 就職活動ジャンルの定義

**就職活動 = 新卒就活特化**
- ES・履歴書の作成ルール
- 面接マナー・対策
- 企業研究・業界分析
- 就活スケジュール・準備

## 作成フロー

### 1. 就活ルール選定
就活本・キャリアセンター情報から確立されたルールを選択

### 2. ルールの明確化
新卒就活に特化した具体的な基準に翻訳

### 3. クイズ化
22文字以内の○×問題として構成

### 4. 解説作成
- わかりやすい解説
- 就活理由・採用側視点を（参考：）で補足

## 作成ルール

### 基本原則
1. **客観的正解**: 就活本・専門家推奨に基づく明確な答え
2. **実用性**: 知らないと不利・知ってると有利な内容
3. **迷いポイント**: 80:20〜50:50の割合で迷いが生じること
4. **22文字制限**: 質問文は全角22文字以内

### 特徴
- 就活生が直面する具体的状況
- 知らないと不利になる情報
- 採用側の視点も含む

### 避けるべき内容
- 転職活動（新卒就活に特化）
- 個人の価値観に依存する選択
- 企業固有の選考プロセス
- 明らかすぎる答え（常識レベル）

## 参考分野一覧

### 応募書類
1. **エントリーシート（ES）**
   - 文字数・構成ルール
   - 効果的な書き方

2. **履歴書**
   - 記載項目・書式
   - 写真・日付のルール

### 面接対策
3. **面接マナー**
   - 入退室の作法
   - 服装・身だしなみ

4. **面接対応**
   - 質問への答え方
   - 逆質問の準備

### 企業研究
5. **業界・企業分析**
   - 情報収集方法
   - 志望動機の作り方

6. **職種理解**
   - 仕事内容の把握
   - 適性の判断

### スケジュール管理
7. **就活スケジュール**
   - 解禁日・選考時期
   - 準備のタイミング

8. **内定・内々定**
   - 返事の期限
   - 辞退のマナー

## クイズ例（完成形）

```
✅【○×】ESの文字数は8割以上埋めるべき
答え：○
解説：ESは文字数の8割以上埋めるのが基本です。空白が多いと熱意が伝わりません。
（参考：就活マナー・ES作成の基本）

✅【○×】面接では最後に「質問はありません」
答え：×
解説：逆質問は必須です。「質問がない」は関心がないと受け取られる可能性があります。
（参考：面接対策・採用担当者の視点）

✅【Q3｜○×】内々定の返事は1週間以内が基本
答え：○
解説：内々定の返事は1週間以内が一般的です。長すぎると企業に迷惑をかけます。
（参考：就活マナー・内定対応）
```

## 品質チェックリスト

- [ ] 22文字以内で質問文が完結している
- [ ] 就活専門書・専門家推奨に基づいている
- [ ] 専門用語を使わずわかりやすい表現
- [ ] 真に迷う要素がある（50:50理想、80:20最低）
- [ ] 実用性が高い
- [ ] 就活・採用の根拠が明記されている
- [ ] 新卒就活に特化し転職活動を含まない
