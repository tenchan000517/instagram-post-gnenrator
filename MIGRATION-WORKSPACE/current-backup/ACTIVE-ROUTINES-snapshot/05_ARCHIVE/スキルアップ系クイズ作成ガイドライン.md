# スキルアップ系クイズ作成ガイドライン

## スキルアップジャンルの定義

**スキルアップ = 学習・能力向上・効率化**
- 学習方法・習得テクニック
- 業務効率化・生産性向上
- 思考法・問題解決
- 自己成長・能力開発

## 作成フロー

### 1. 理論選定
学習理論・認知科学・行動科学から客観的根拠のあるものを選択

### 2. 理論のかみ砕き
専門用語を使わず、誰でも理解できる身近な表現に翻訳

### 3. クイズ化
22文字以内の○×問題として構成

### 4. 解説作成
- わかりやすい解説
- 学問的根拠を（参考：）で補足

## 作成ルール

### 基本原則
1. **客観的正解**: 理論・研究に基づく明確な答えがあること
2. **実用性**: 知らないと損する・知ってると役立つ内容
3. **迷いポイント**: 80:20〜50:50の割合で迷いが生じること
4. **22文字制限**: 質問文は全角22文字以内

### 特徴
- 実践的で即効性がある内容
- 仕事・勉強に直結する知識
- 科学的根拠に基づく方法論

### 避けるべき内容
- 転職・就職の具体的手法
- 会社特有のルール・文化
- 主観的な価値判断
- 明らかすぎる答え（常識レベル）

## クイズ例（完成形）

```
✅【○×】復習は翌日より1週間後が効果的
答え：×
解説：翌日の復習が最も効果的です。忘却曲線に基づくと、24時間以内の復習が記憶定着に重要。
（参考：エビングハウスの忘却曲線）

✅【○×】一度に覚えられるのは7±2個まで
答え：○
解説：ワーキングメモリの容量は約7個です。それ以上は覚えにくくなります。
（参考：ミラーのマジカルナンバー7）

✅【○×】重要な仕事の80%は20%の時間で終わる
答え：○
解説：パレートの法則では、成果の80%は20%の要因から生まれると言われています。
（参考：パレートの法則）
```
