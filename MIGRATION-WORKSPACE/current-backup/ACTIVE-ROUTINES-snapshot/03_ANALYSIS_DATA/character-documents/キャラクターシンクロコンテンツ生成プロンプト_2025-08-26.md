## 一括生成プロンプト（複数コンテンツ対応）

```
**一括変換指示**
以下の元ソース5個を、指定キャラクター（IIDA/MISAKI/KING）の視点で一括変換してください。

**キャラクター**: [IIDA/MISAKI/KINGから指定]

**元ソース一覧**:
1. [元ソース1]
2. [元ソース2]
3. [元ソース3]
4. [元ソース4]
5. [元ソース5]

**変換ルール**:
該当キャラクターのシンクロドキュメントに基づき、各コンテンツを完全に変換してください。

**出力形式**:
## コンテンツ1
[変換結果1]

## コンテンツ2
[変換結果2]

（以下同様）

**品質チェック**:
各コンテンツについて以下を確認してください
- そのキャラクターらしい自然な語り口になっているか
- ターゲット読者に適した表現になっているか
- 曖昧な指示語や不自然な表現がないか
- 断られやすい表現（〜ませんか）を使っていないか
```

## 品質チェック用プロンプト

```
**品質評価指示**
以下のコンテンツが適切にキャラクター変換されているか評価してください。

**評価対象コンテンツ**: [変換後コンテンツを挿入]
**キャラクター**: [IIDA/MISAKI/KING]
**元ソース**: [元ソースを挿入]

**評価基準**:
1. **キャラクター整合性** (10点満点)
   - そのキャラクターらしい語り口になっているか
   - キャラクターの専門知見が自然に活かされているか

2. **ターゲット適合性** (10点満点)
   - 対象読者（学生/女性/社会人）に適した表現か
   - 不適切な表現（年代・立場ミスマッチ）がないか

3. **日本語の自然さ** (10点満点)
   - 敬語レベルの一致
   - 不自然な語尾・表現がないか

4. **有益性** (10点満点)
   - 実際に読者の役に立つ内容か
   - 具体的で実践的か

5. **機能性** (10点満点)
   - フックとして機能するか
   - 行動促進になっているか

**出力形式**:
- 各項目の点数と理由
- 改善すべき具体的な箇所
- 修正案の提示
```

## 使用方法

### ステップ1: 事前準備
キャラクターシンクロドキュメントを読み込み、各キャラクターの思考回路を理解

### ステップ2: プロンプト選択
- 単体生成: 各キャラクター別プロンプトを使用
- 一括生成: 一括生成プロンプトを使用
- 品質確認: 品質チェック用プロンプトを使用

### ステップ3: 実行
元ソースを挿入してプロンプトを実行

### ステップ4: 品質確認
生成結果を品質チェック用プロンプトで評価・改善
