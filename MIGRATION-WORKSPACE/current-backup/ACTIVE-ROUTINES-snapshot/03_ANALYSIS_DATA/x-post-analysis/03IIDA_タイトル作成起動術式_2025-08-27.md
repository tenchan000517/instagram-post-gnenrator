# IIDA タイトル作成起動術式

作成日: 2025-08-27  
対象: IIDAキャラクター専用タイトル作成  
レベル: 次世代Claude Code用完全版

---

## 🚀 起動コマンド

```
Claude、IIDAタイトル作成マニュアルに従って、
社会人向けコンテンツ100個02のタイトル集を作成してください。

【作業フロー】
1. コンテンツを読み込み（3回熟読）
2. 成功定義を明確化（就活生・新卒・若手が「成長したい」「改善したい」と思うこと）
3. 就活・キャリア特化タイトル作成（「誰のこと」が一瞬でわかる）
4. 4タイプ作成（+5選、-÷対比、×リンク）
5. 自己評価とIIDAらしさチェック
```

---

## ⚡ 高速作成チェックリスト

### STEP1: コンテンツ理解（IIDA特化）
- [ ] コンテンツの本質を就活・キャリア文脈で把握
- [ ] 就活生・新卒・若手に何を感じてほしいか明確化
- [ ] 元コンテンツの言葉を就活・キャリア用語で解釈

### STEP2: タイトル作成
- [ ] **+タイプ（必ず5選）**: 就活・キャリア失敗型 or 実践指導型
- [ ] **-タイプ（対比用）**: うまくいかない人・成功しない人
- [ ] **÷タイプ（対比用）**: うまくいく人・成功する人（-と必ず対比）
- [ ] **×タイプ（リンク）**: +の内容に合わせた具体的解決法

### STEP3: IIDAらしさチェック
- [ ] 誠実で指導的な表現（「改善が必要」「注意すべき」）
- [ ] 実践的・段階的なアプローチ感
- [ ] 就活生・新卒・若手に刺さるか

### STEP4: NG回避チェック
- [ ] 攻撃的すぎる表現（「残念な」「ダメな」）使用していない
- [ ] 感情的すぎる表現避けている
- [ ] 就活・キャリア以外のジャンル混入なし
- [ ] 18-28歳の実際の悩みに対応しているか

## 💡 作成例（コンテンツ1）

### 元コンテンツの本質（IIDA解釈）
「君には可能性がある、挑戦して成長しろ」
→ 「就活・キャリアで挑戦することの重要性」

### 成功定義（IIDA特化）
就活生・新卒・若手が「挑戦してみよう」「成長したい」と思うこと

### タイトル4タイプ（IIDA版）
**+タイプ**: 「成長が止まる人の特徴5選」
**-タイプ**: 「挑戦を避ける人」
**×タイプ**: 「着実に成長するための方法」
**÷タイプ**: 「継続的に成長する人」

## 🎯 効果的なパターン集（IIDA特化）

### 就活失敗型（推奨）
```
「面接で落ちる人の特徴5選」
「内定をもらえない人の特徴5選」
「ES通過しない人の特徴5選」
```

### 新卒・若手失敗型（推奨）
```
「新入社員で嫌われる人の特徴5選」
「先輩に評価されない人の特徴5選」
「職場で浮く人の特徴5選」
```

### キャリア停滞型（推奨）
```
「キャリアが伸び悩む人の特徴5選」
「転職で失敗する人の特徴5選」
「成長が止まる人の特徴5選」
```

### IIDA特有の指導的表現
```
「○○で注意すべき人の特徴5選」
「○○で改善が必要な人の特徴5選」
「○○でつまずく人の特徴5選」
```

### 実践指導型
```
「○○を成功させる方法5選」
「○○で評価される方法5選」
「○○を身につける手順5選」
```

## 🔥 IIDA専用指導フレーズ

タイトル作成時の心構え：
- 「一緒に頑張りましょう」感を込める
- 読み手の成長を真剣に支援する姿勢
- 段階的で実践的なアドバイス提供

---

## 📚 IIDA特化表現ガイド

### 推奨表現
- 「うまくいかない」「成功しない」「つまずく」
- 「改善が必要」「注意すべき」「見直すべき」
- 「身につける」「習得する」「成長する」
- 「着実に」「段階的に」「計画的に」

### 避ける表現
- 「残念な」「ダメな」「致命的な」（KING的すぎる）
- 「〜かもしれません」（MISAKI的すぎる）
- 就活・キャリア以外の専門用語
- 感情論・精神論的な表現

---

## ⚠️ 最重要注意（IIDA特化）

**我々は就活・キャリア特化アカウント**
→ 就活生・新卒・若手の「リアルな悩み」に直結しないタイトルは絶対NG
→ 読み手が「これ、私のことだ」と感じられることが必須

**年齢層特化**
→ 18-28歳の実際の課題・悩みに基づく
→ 30代以上向けの内容は避ける

---

## 💼 IIDA専用成功パターン

### 就活段階
- 面接、ES、企業研究、自己分析関連
- 就活準備、選考対策、内定獲得関連

### 新卒段階
- 職場適応、先輩関係、業務習得関連
- ビジネスマナー、人間関係構築関連

### 若手段階
- スキル習得、キャリア形成、転職関連
- 成長実感、将来設計、目標達成関連
