# MISAKI起動術式

作成日: 2025-08-26  
対象: MISAKIキャラクター専用タイトル作成

---

## 起動コマンド

**指定したコンテンツファイルから5～10個のコンテンツのタイトルを作成してください。**

**使用マニュアル**: `01MISAKI_タイトル作成マニュアル_2025-08-26.md` を参照

**対象ファイル**: 女性向けコンテンツ100個○○.md

**作成範囲**: コンテンツ○番～○番（5～10個指定）

---

## 実行フロー

1. **コンテンツ読み込み**: 指定ファイルの指定範囲を3回熟読
2. **成功定義**: 各コンテンツで「視聴者にどうなってほしいか」を考える  
3. **事例参照**: マニュアルのSランク・Aランク事例を読み込む
4. **タイトル作成**: 各コンテンツの+-×÷を作成（5～10選）
5. **自己評価**: 頻出チェック・事例比較チェック
6. **修正**: 必要に応じて修正
7. **出力**: 指定形式で出力

---

## 出力形式

```

## コンテンツ○: ○○○○○

（以下、指定範囲まで継続）
```

---

## 使用例

**コマンド例**:
「女性向けコンテンツ100個02.mdのコンテンツ7～10のタイトルを作成してください。」

**実行**: 
- 01MISAKI_タイトル作成マニュアル_2025-08-26.md を参照
- フロー1～7を実行
- 上記出力形式で結果出力
