# IIDA リール動画作成マニュアル

**作成日**: 2025-08-27  
**目的**: IIDAコンテンツのリール動画作成プロセスの体系化  
**対象**: 就活・新卒・若手社会人向けコンテンツ特化

---

## 📋 基本構成（10ページ固定）

1. **1ページ目**: メインタイトル（+タイプ）
2. **2ページ目**: 元コンテンツそのまま掲載
3. **3ページ目**: 比較表（-タイプvs÷タイプ）
4. **4ページ目**: メインタイトル再掲
5. **5-9ページ目**: 特徴説明5つ（問題→改善アクション→結果）
6. **10ページ目**: CTA（固定・作成不要）

---

## 🎯 コンテンツ連動構造

### ファイル連携
- **元コンテンツ**: `社会人向けコンテンツ100個02.md`（激励メッセージ）
- **タイトル集**: `コンテンツタイトル02.md`（4タイプのタイトル）

### 各タイプの役割
- **+タイプ**: メインタイトル（5選固定・就活キャリア系）
- **-タイプ**: ネガティブ比較（8選固定）  
- **×タイプ**: 実践コツの裏テーマ（5選固定）
- **÷タイプ**: ポジティブ比較（8選固定）

---

## ✅ 3ページ目：比較表作成ルール

### 必須要件
- 比較対象タイトル12文字以内（例：面接で緊張する人 / 面接で評価される人）
- 各項目12文字以内
- 「〇〇な人の特徴8選」で統一
- 左右で正確な対比構造
- **番号なし（1.2.3.を使用しない）**
- **各項目間に空行挿入**

### 品質基準（IIDA特化）
- 就活・新卒・若手のリアルな課題
- 実践的で具体的な内容
- 指導的で建設的な表現

### 完成例フォーマット
```
面接で緊張する人
面接で評価される人

面接で緊張する人の特徴8選

準備が不十分

質問の意図を理解していない

自己分析が浅い

企業研究が足りない

...

面接で評価される人の特徴8選

事前準備を徹底している

質問に的確に答える

自分の強みを理解している

企業への理解が深い

...
```

## 🚀 5-9ページ目：特徴説明ページ作成の極意

### IIDA流の構成
**特徴タイトル → 問題の説明 → 改善アクション → 結果**

- 各ページに1つの特徴を詳しく説明
- 「していた」過去形で過去の問題を表現
- 現在の改善アクションと結果を明示

### 元コンテンツとの繋がり（必須）
1. **段階的に**スキル習得に取り組む（実践的）
2. **計画的に**キャリア形成を進める決断
3. **早い段階で**正しい方向に軌道修正
4. 結果として着実なキャリア成長を得る

### 各ページの構成パターン
```
### 〇ページ目: 特徴〇
[具体的な問題行動]していた

「[セリフ形式の問題描写]」  
[問題の本質的な説明]  
今は[改善アクション]  
[具体的な成果・結果]
```

### 使うべき表現（IIDA特化）
- 問題描写：「していた」過去形で統一
- セリフ形式：「〜がわからない」「〜が不安」など
- 改善表現：「今は〜する」現在形
- 結果表現：具体的な成果を明示

### エピソード作成のコツ（IIDA特化）
1. **実用性**: 就活・キャリアで実際に使える内容
2. **段階性**: ステップバイステップのアプローチ
3. **具体性**: 行動と結果を明確に
4. **共感性**: 18-28歳の実際の悩みに対応

### 文体の注意点（IIDA特化）
- 誠実で指導的な文章
- 実践的でわかりやすい説明
- 「改善が必要」「注意すべき」などの建設的表現
- IIDAらしい真面目で親身な指導

### コピペ対応の重要ポイント
- **リール動画構成部分は装飾記号を一切使用しない**
- **（太字）、左：右：などの記号は削除**
- 対比の1. 2. などの番号付きリストも削除
- コピーしてそのまま投稿に使える形式を徹底

## 📝 IIDAキャラクター特性

### 使う表現
- 指導的な言葉（「改善が必要」「注意すべき」「見直すべき」）
- 実践的なアドバイス（「具体的には〜」「実際に〜」）
- 「一緒に頑張りましょう」「着実に成長できます」
- 就活・キャリア用語・成長重視の表現

### 使わない表現
- 攻撃的すぎる表現（KINGの「残念な」「ダメな」）
- 感情的な表現（MISAKIの「〜かもしれません」）
- 曖昧な精神論
- 就活・キャリア以外のジャンル表現

---

## 🔄 作成プロセス

1. **元コンテンツ確認**: 激励メッセージの核心を理解
2. **タイトル確認**: 4タイプの役割と内容を把握
3. **比較表作成**: 就活・キャリアの実際の課題対比を作成
4. **特徴ページ作成**: 問題→改善→結果を5つ作成
5. **全体調整**: 改行・文字数・表現の統一

### キャプション構成ルール（IIDA用）
1. IIDAキャラクター別掴み文（1-2文）
2. 構造化された投稿補足説明（15行程度）
3. **固定コミュニティ誘導文**（変更厳禁）
4. 固定フォロー促進文
5. ハッシュタグ（12個・IIDA推奨パターン使用）

## 📌 改善ポイント

### よくある失敗（IIDA特化）
- 元コンテンツとの繋がりが薄い
- 指導的でなく感情的すぎる表現
- エピソードが抽象的で実践性不足
- 就活・キャリア以外の要素混入
- 年齢層に合わない内容

### 成功のコツ（IIDA特化）
- 元コンテンツの激励を就活・キャリア文脈で理解
- 「私も同じ悩みを抱えていました」と共感できる内容
- 具体的で段階的な改善アクションを提示
- IIDAの誠実で実践的なキャラクターを貫く
