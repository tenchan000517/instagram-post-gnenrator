## STEP 1: コンテンツ読み込み

対象コンテンツファイル（社会人向けコンテンツ100個02.md）から指定されたコンテンツを3回熟読する。

**注意**: ファイルが複数存在する可能性があるため、どのファイルのどのコンテンツかを明確にする。

## STEP 2: 成功定義

**質問**: このコンテンツで就活生・新卒・若手社会人に「どうなってほしいか」？

**IIDA向け成功定義の特徴**:
- 就活・転職での成功
- キャリア選択の明確化
- 面接・ES・自己PR力向上
- 職場適応・人間関係構築
- 実践的なスキル習得
- 将来への不安解消

**例**:
- コンテンツ1（困難な環境で成長）→ 就活・新卒での挑戦に前向きになる
- コンテンツ4（目の前の仕事に集中）→ 新入社員として集中力を発揮する
- コンテンツ12（自分軸の確立）→ キャリア軸を明確にして就活に臨む

## STEP 4: タイトル作成（4タイプ）

### **+タイプ（メインタイトル）**

#### **基本ルール**
- **固定語尾**: 「〜の特徴5選」で統一
- **文字数**: 14文字程度（Instagram表示最適化）
- **ターゲット明確化**: 就活生・新卒・若手にピンポイント

#### **IIDA特化パターン**
```
パターン1: 就活失敗系
「内定をもらえない人の特徴5選」
「面接で落ちる人の特徴5選」
「ES通過しない人の特徴5選」

パターン2: 新卒・若手失敗系
「新入社員で嫌われる人の特徴5選」
「先輩に評価されない人の特徴5選」
「職場で浮く人の特徴5選」

パターン3: キャリア停滞系
「キャリアが伸び悩む人の特徴5選」
「転職で失敗する人の特徴5選」
「成長が止まる人の特徴5選」
```

#### **表現ルール**
- **真面目で誠実**: 「残念な」より「うまくいかない」「成功しない」
- **指導的**: 「致命的」より「注意すべき」「改善が必要な」
- **実践的**: 具体的で実用的な表現を重視

### **-タイプ（ネガティブ比較）**

#### **基本方針**
- +タイプの問題を抱える人を12文字以内で表現
- 就活・キャリアの「つまずきポイント」に特化

#### **IIDA特化パターン**
```
就活系: 「面接で緊張する人」「自己分析ができない人」「企業研究不足の人」
新卒系: 「職場に慣れない人」「先輩とうまくいかない人」「仕事が覚えられない人」
キャリア系: 「将来が不安な人」「スキルが身につかない人」「転職を迷う人」
```

### **×タイプ（実践コツの裏テーマ）**

#### **基本方針**
- 具体的な解決策・実践法を5選で提示
- IIDAらしい実用的で段階的なアプローチ

#### **IIDA特化パターン**
```
就活系: 「面接突破の準備法」「ES作成の具体的ステップ」「企業研究の効率的方法」
新卒系: 「職場適応の実践法」「先輩との関係構築法」「業務習得の効果的手順」
キャリア系: 「スキル習得の計画的方法」「転職成功の準備手順」「キャリアプラン作成法」
```

### **÷タイプ（ポジティブ比較）**

#### **基本方針**
- -タイプと正反対の「成功している人」を12文字以内で表現
- 就活・キャリアで「うまくいく人」の特徴

#### **IIDA特化パターン**
```
就活系: 「内定をもらう人」「面接で評価される人」「志望企業に受かる人」
新卒系: 「職場で信頼される人」「先輩に可愛がられる人」「仕事を早く覚える人」
キャリア系: 「将来が明確な人」「着実に成長する人」「転職で成功する人」
```

## STEP 5: 自己評価チェック

### **チェック項目**

#### **IIDA特化品質基準**
- [ ] 就活・新卒・若手のリアルな悩みに直結しているか？
- [ ] 実践的で具体的なアドバイスにつながるか？
- [ ] IIDAらしい誠実で指導的な表現か？
- [ ] ターゲット年齢層（18-28歳）に適合しているか？

#### **基本品質基準**
- [ ] +タイプは「〜の特徴5選」で統一されているか？
- [ ] -と÷タイプは明確に対比関係にあるか？
- [ ] ×タイプは+タイトルと内容連動しているか？
- [ ] 文字数制限（+14文字、-÷12文字）は守られているか？

#### **頻出ワード回避チェック**
- [ ] 「やってる」「人の」「ヤバい」「筋トレ」未使用
- [ ] 「瞬間」「証拠」「理由」などの不自然表現回避
- [ ] 就活・キャリア分野以外のワード混入なし

---

## STEP 6: 修正

チェックで問題があった場合の修正方針：

### **よくある修正パターン**
1. **ターゲット外れ**: 就活・キャリア以外 → 就活・新卒・若手向けに調整
2. **表現が攻撃的**: KINGの攻め表現 → IIDAの指導的で誠実な表現
3. **文字数オーバー**: 略語使用、助詞削減で調整
4. **対比関係弱い**: -と÷の関係を明確な正反対に調整
