# MISAKIタイトル作成マニュアル

作成日: 2025-08-26  
対象: MISAKIキャラクター専用  
範囲: コンテンツ読み込み～タイトル確定（フロー1-7）

---

## フロー概要

1. **コンテンツ読み込み**
2. **成功定義**（コンテンツから「成功＝何か」を考える）
3. **事例参照**（Sランク・Aランク事例集を読み込む）
4. **タイトル作成**（+-×÷の4つ作成）
5. **自己評価**（頻出チェック・事例比較チェック）
6. **修正**（必要に応じて）
7. **出力**（タイトル4つ確定）

---

## STEP 1: コンテンツ読み込み

対象コンテンツファイル（例：女性向けコンテンツ100個01.md）から指定されたコンテンツを3回熟読する。

**注意**: ファイルが複数存在（01.md、02.md、03.md...）するため、どのファイルのどのコンテンツかを明確にする。

---

## STEP 2: 成功定義

**質問**: このコンテンツで視聴者に「どうなってほしいか」？

**例**:
- コンテンツ1（楽な道vs挑戦の道）→ 小さな挑戦を始める勇気を持つ
- コンテンツ6（仕事80%の力）→ 余裕を持って仕事する（定時退社）

## STEP 4: タイトル作成

### +-×÷の4タイプを作成

**+タイプ**: ゴールに向かう方法・行動・習慣
**-タイプ**: ゴールに到達できない人の思考・行動
**×タイプ**: ゴールのための具体的メソッド・手法
**÷タイプ**: ゴールに既に到達した人の状態

### 数字の範囲
5～10選の範囲で設定

---

## STEP 5: 自己評価

### 頻出チェック（マスト）
- [ ] 同じ表現を使いすぎていないか
- [ ] 前のコンテンツで使った表現と被っていないか

**NGパターン例**:
- 全部「思考パターン」
- 全部「特徴」
- 全部「やりがちなこと」

### 事例比較チェック
- [ ] Sランク事例と比べて突飛になっていないか
- [ ] ワードセンスが自然か
- [ ] 日常的に使う表現になっているか

### 📋 高品質チェック（フィードバック反映版）

#### レベル1：基本ワードセンス
- [ ] **二重表現チェック**：「〜作り6つの方法」等の重複表現なし
- [ ] **抽象語チェック**：「術」「システム」等の硬い表現なし
- [ ] **自然さチェック**：日常会話で使える表現になっている

#### レベル2：表現バリエーション
- [ ] **「方法」多用チェック**：「コツ」「メソッド」でバリエーション作成
- [ ] **視覚的表現活用**：「見る景色」「見る世界」で想像しやすい
※ただし使用には文脈とセンスが必要なので慎重に使うこと
- [ ] **記号活用**：「○」「◎」でカジュアル感演出
※ただし使用には文脈とセンスが必要なので慎重に使うこと

#### レベル3：コンテンツ理解
- [ ] **本質理解チェック**：コンテンツの真意を正確に反映
- [ ] **文脈適合チェック**：単語の一般的意味に惑わされていない
- [ ] **MISAKIらしさチェック**：キャラクターに適した温かみのある表現

#### レベル4：高品質化
- [ ] **Sランク事例活用**：「1日1%の小さな習慣」等の直接参照
- [ ] **具体性強化**：「3年続けたら」「1日1％」等の期間・数字明示
※ただし文脈をよく考えることと虚偽の数字を使わないこと
- [ ] **体験重視**：「〜した瞬間」「〜の体験談」で臨場感
- [ ] **親しみやすさ最優先**：理論より人間味重視
