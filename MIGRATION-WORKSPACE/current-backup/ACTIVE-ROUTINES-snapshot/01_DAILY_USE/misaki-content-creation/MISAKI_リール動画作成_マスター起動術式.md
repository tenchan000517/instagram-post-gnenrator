## 🛠️ 作成手順（5ステップ）

### **STEP1: 基盤情報収集**
- [ ] 指定されたコンテンツ番号の元コンテンツを読み取り
- [ ] 対応する4タイプタイトルを確認
- [ ] 元コンテンツの核心メッセージを理解

### **STEP2: 10ページ構成設計**
**基本構成（002完全準拠）**:
```
1ページ目: メインタイトル
2ページ目: 元コンテンツ（MISAKIらしい優しい言葉に変換）
3ページ目: 比較表（リスト形式）
4ページ目: 改めてタイトル
5-9ページ目: コツページ（+タイトルの語尾に応じた形式）
10ページ目: CTA（固定・作成不要）
```

### **STEP3: 比較表作成**
**形式**:
```
例：目標が達成できない人
例：目標達成した人

[−タイプタイトル]
項目1

項目2

...項目8
（各項目の間に空行を入れる）

[÷タイプタイトル]
項目1

項目2

...項目8
（各項目の間に空行を入れる）

```

**作成規則**:
- [ ] -タイプと÷タイプから8選ずつ抽出
- [ ] 各項目12文字以内で調整
- [ ] 女性らしいあるある感を確保
- [ ] 正確な対比構造を維持

### **STEP4: コツページ作成（5-9ページ）**

#### **+タイトル語尾別対応**
```
「〜したこと」   → 結果・体験談形式
「〜した行動」   → 具体的な行動・アクション形式  
「〜した方法」   → 手法・やり方形式
「〜した習慣」   → 習慣・ルーティン形式
```

#### **ページ構成テンプレート**
```
### Xページ目: +タイトル+習慣X
[簡潔なタイトル]（12文字以内目安）

[説明文4行]
- 1行目: 状況設定（「〜と思いますが」等）
- 2行目: 行動提案（「〜してみると」等）
- 3行目: 結果・効果（「〜できて」等）
- 4行目: 追加効果（「〜します」等）
```

#### **重要な作成規則**
- [ ] 元コンテンツのテーマと×タイプの掛け算を実現
- [ ] 仕事での挑戦に統一（一貫性確保）
- [ ] 意図的・能動的なエピソードに調整
- [ ] MISAKIらしい温かい表現で統一
- [ ] **体験談の一般化**: 「〜した」→「〜します」「〜できます」
- [ ] 改行・文字数を統一（コピペしやすく）

### **STEP5: キャプション生成・保存**
- [ ] MISAKI用共感寄り添い型掴み文を使用
- [ ] 女性向けハッシュタグセットを適用
- [ ] コピペしやすいフォーマットに調整
- [ ] ファイルを指定形式で保存: `misaki_content_XXX.md`

## 📏 フォーマット仕様（002完全準拠）

### **ページタイトル形式**
```
### 1ページ目: メインタイトル
### 2ページ目: 元コンテンツ
### 3ページ目: 比較表
### 4ページ目: 改めてタイトル
### 5ページ目: [+タイトル]+習慣1
### 6ページ目: [+タイトル]+習慣2
...
### 10ページ目: CTA
```

### **コツページタイトル最適化ルール**
```
【簡潔化4原則】
1. 修飾語削除：「難しい」「チャレンジングな」等除去
2. 動詞直接化：「手を挙げる」→「挑戦する」
3. 主語省略：「上司に」→省略（文脈で理解可能）
4. 10-12文字以内：読みやすさ重視

❌ 冗長例: 「チャレンジングなプロジェクトに自分から手を挙げる習慣」
✅ 最適例: 「プロジェクトに挑戦する」

❌ 冗長例: 「毎朝30分早起きして目標に向かう準備を整える習慣」
✅ 最適例: 「毎朝30分早起きする」
```

### **体験談一般化変換例**
```
❌ 個人体験: 「最初は不安もあったけど思い切って志願してよかった」
✅ 一般化: 「最初は不安もあると思いますが思い切って志願してみると」

❌ 個人体験: 「新しいスキルが身についたし上司からも信頼された」
✅ 一般化: 「新しいスキルが身につくし上司からも信頼されます」
```

## 🔧 トラブルシューティング

### **よくある失敗と対処法**

**問題**: +タイトルと5-9ページの内容が不一致
**対処**: 「〜したこと」「〜した行動」「〜した方法」「〜した習慣」の語尾を正確に把握し、内容を合わせる

**問題**: 元コンテンツとの繋がりが薄い
**対処**: 元コンテンツの核心メッセージを5つのエピソード全てに反映

**問題**: 体験談が個人的すぎる
**対処**: 「〜した」→「〜します」「〜と思います」に変更し、一般化

**問題**: エピソードがワンパターン
**対処**: 人間関係・スキル・評価・視野・楽しさの5つの観点で差別化

**問題**: タイトルが長すぎる
**対処**: 12文字以内を目安に、核心部分のみに絞る
