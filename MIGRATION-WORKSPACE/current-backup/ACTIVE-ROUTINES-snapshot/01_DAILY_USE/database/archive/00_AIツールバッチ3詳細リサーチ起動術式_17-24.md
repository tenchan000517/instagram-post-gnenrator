## ⭐ TEN評価軸による絶対評価（必須）

**重要**: 各ツールについて以下のTEN評価軸で100点満点の**絶対評価**を実行してください：

```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### **評価項目詳細：**
1. **即効性(20%)**: どれだけ早く結果が得られるか（0-100点）
2. **簡単さ(20%)**: 学習コストの低さ・操作の直感性（0-100点）  
3. **人気度(15%)**: ユーザー数・市場での認知度（0-100点）
4. **コスパ(15%)**: 価格に対する機能・価値（0-100点）
5. **機能専門性(20%)**: 独自機能・差別化要素の強さ（0-100点）
6. **生産性UP度(10%)**: 実際の作業効率向上効果（0-100点）

### **evaluation項目の必須記載：**
```json
"evaluation": {
  "immediacy": 85,           // 即効性（0-100点）
  "simplicity": 80,          // 簡単さ（0-100点）
  "popularity": 95,          // 人気度（0-100点）
  "costPerformance": 70,     // コスパ（0-100点）
  "specialization": 90,      // 機能専門性（0-100点）
  "productivityGain": 85,    // 生産性UP度（0-100点）
  "tenScore": 84.25          // 総合TENスコア
}
```

**注意**: rank項目は不要です（バッチ内ランキングは無意味のため）
