**生産性ソフト詳細情報の完全収集・構造化**
- **対象**: TENランキング上位5-10項目（一括処理）

- **重点情報**:

- **品質基準**: 実際に導入・活用可能なレベルの詳細情報

## 📋 詳細リサーチ手順（5段階プロセス）

### **🔍 STEP 1: 基本情報詳細収集**

各生産性ソフトについて以下の基本情報を詳細収集：

#### **検索クエリ例**
```
"[ツール名] 2025 features pricing plans comparison"
"[ツール名] free vs premium limitations workflow"
"[ツール名] Japanese support localization quality"
"[ツール名] setup guide tutorial getting started"
"[ツール名] team collaboration features enterprise"
```

#### **収集必須情報**
- **正式名称・開発会社・企業情報**
- **サービス開始年・最新アップデート状況**
- **料金体系（個人・チーム・企業プラン詳細）**
- **日本語対応レベル（UI・サポート・ドキュメント・データ移行）**
- **対応プラットフォーム（Web・モバイル・デスクトップ）**
- **ユーザー数・企業導入実績**

---

### **🎮 STEP 2: 実用性・ワークフロー詳細調査**

#### **検索クエリ例**
```
"[ツール名] workflow integration productivity review 2025"
"[ツール名] real world use cases business examples"
"[ツール名] setup tutorial best practices workflow"
"[ツール名] team adoption learning curve experience"
"[ツール名] automation features time saving benefits"
```

#### **収集必須情報**
- **主要用途・業務シーン（5-7個の具体例）**
- **学習コスト・チーム導入期間**
- **ワークフロー統合・他ツール連携**
- **実際の業務効率化効果・時短効果**
- **チーム利用時のコラボレーション機能**
- **カスタマイズ・拡張機能**

---

### **💰 STEP 3: 料金・機能制限詳細分析**

#### **検索クエリ例**
```
"[ツール名] pricing 2025 free tier limitations users"
"[ツール名] cost comparison value for money ROI"
"[ツール名] enterprise pricing team size scaling"
"[ツール名] free alternatives open source comparison"
```

#### **収集必須情報**
- **無料版の具体的制限（ユーザー数・ストレージ・機能等）**
- **有料版プラン詳細・価格・機能差**
- **企業向けプランの要件・価格**
- **スケールアップ時のコスト変化**
- **コストパフォーマンス評価・ROI**
- **予算制約時の代替案・オープンソース選択肢**

---

### **⚔️ STEP 4: 競合比較・差別化分析**

#### **検索クエリ例**
```
"[ツール名] vs [競合1] vs [競合2] 2025 comparison"
"[ツール名] competitive advantages unique features"
"[ツール名] market position strengths weaknesses"
"[ツール名] migration from [競合ツール] guide"
```

#### **収集必須情報**
- **直接競合ツール（3-5個）とカテゴリ内位置**
- **独自機能・差別化ポイント**
- **他ツールに対する明確な優位性**
- **劣る点・課題・改善要望**
- **選択理由・推奨条件・適用場面**

---

### **📊 STEP 5: TEN評価軸詳細分析**

最終的に各項目のTEN評価を詳細根拠と共に確定：

#### **修正TEN評価軸（6項目）**
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

#### **各軸の詳細根拠収集（生産性ソフト特化）**
- **即効性**: 導入から業務効率化実感までの具体的期間・事例
- **簡単さ**: 実際の設定手順・学習コスト・チーム展開の容易さ
- **人気度**: 企業導入数・業界シェア・転職での評価度
- **コスパ**: 価格対効果・無料版実用性・スケールコスト
- **機能専門性**: 業務特化機能・独自性・技術的優位性
- **生産性UP度**: 実際の時短効果・業務品質向上・チーム効率化

## 🎯 品質チェックリスト

### 必須確認項目
- [ ] **実用性検証**: 実際に導入可能なレベルの詳細情報
- [ ] **最新情報**: 2025年1月以降の最新機能・価格情報反映
- [ ] **日本語対応**: UI・サポート・移行支援の実際の品質
- [ ] **無料版詳細**: 個人・小規模チーム利用での制限・実用性
- [ ] **競合比較**: 同カテゴリ内での客観的相対評価
- [ ] **業務効率化効果**: 具体的な時短・効率化効果の事例
- [ ] **チーム導入配慮**: 組織導入時の課題・ベストプラクティス
- [ ] **統合性評価**: 既存ワークフローへの統合容易性

### 絶対避けるべき項目
- [ ] **古い情報**: 2024年以前の機能・価格での判断
- [ ] **机上の空論**: 実際の業務に適用困難な理論的評価
- [ ] **過大効果**: 現実的でない業務改善効果の誇張
- [ ] **コスト見落とし**: 隠れコスト・スケールコスト無視
- [ ] **競合無視**: 単独評価での偏った評価
