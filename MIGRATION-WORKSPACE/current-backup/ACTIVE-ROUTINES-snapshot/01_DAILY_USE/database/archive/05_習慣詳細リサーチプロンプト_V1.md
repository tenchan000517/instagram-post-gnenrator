## 🎯 詳細リサーチ目的

**生産性習慣詳細情報の完全収集・構造化**
- **対象**: TENランキング上位5-10項目（一括処理）
- **目標**: Instagram投稿用の完全データセット作成
- **重点情報**: 
  - 科学的根拠・研究データの詳細
  - 実践方法・継続のコツの具体的情報
  - 実際の効果・体感変化の事例
  - 習慣化にかかる期間・挫折要因
  - 他習慣との組み合わせ効果
  - 忙しい人向けの簡易実践法
- **品質基準**: 実際に習慣化可能なレベルの詳細情報

---

## 📋 詳細リサーチ手順（5段階プロセス）

### **🔍 STEP 1: 科学的根拠・研究データ詳細収集**

各習慣について以下の基本情報を詳細収集：

#### **検索クエリ例**
```
"[習慣名] scientific research studies benefits 2025"
"[習慣名] neuroscience productivity brain research"
"[習慣名] psychology behavior change effectiveness"
"[習慣名] peer reviewed studies evidence meta analysis"
"[習慣名] Harvard Stanford research findings 2024 2025"
```

#### **収集必須情報**
- **科学的根拠・研究論文・エビデンス**
- **脳科学・心理学・生理学的メカニズム**
- **効果の定量的データ（改善率・パフォーマンス向上等）**
- **研究機関・大学での実験結果**
- **メタ分析・大規模調査の結果**
- **効果の個人差・条件依存性**

---

### **🎮 STEP 2: 実践方法・継続戦略詳細調査**

#### **検索クエリ例**
```
"[習慣名] how to start implementation guide beginner"
"[習慣名] habit formation tips consistency strategy"
"[習慣名] busy people quick implementation method"
"[習慣名] common mistakes obstacles how to overcome"
"[習慣名] habit stacking combination with other habits"
```

#### **収集必須情報**
- **段階的実践方法（初心者→上級者の道筋）**
- **必要時間・頻度・実践環境**
- **継続のコツ・モチベーション維持法**
- **よくある挫折要因・対処法**
- **他習慣との組み合わせ・相乗効果**
- **忙しい人向けの最小実施法**

---

### **💰 STEP 3: コスト・投資要件詳細分析**

#### **検索クエリ例**
```
"[習慣名] cost implementation tools equipment needed"
"[習慣名] free vs paid methods budget comparison"
"[習慣名] time investment daily weekly schedule"
"[習慣名] ROI productivity return on time investment"
```

#### **収集必須情報**
- **実践に必要なコスト（道具・アプリ・環境等）**
- **時間投資・日常スケジュールへの影響**
- **無料実践 vs 有料ツール活用の効果差**
- **長期継続でのコストパフォーマンス**
- **時間対効果・投資回収期間**
- **予算制約時の代替実践法**

---

### **⚔️ STEP 4: 効果比較・適用場面分析**

#### **検索クエリ例**
```
"[習慣名] vs [類似習慣1] vs [類似習慣2] effectiveness"
"[習慣名] personality type MBTI compatibility effectiveness"
"[習慣名] work style remote office effectiveness"
"[習慣名] age group demographics effectiveness research"
```

#### **収集必須情報**
- **類似習慣・代替手法（3-5個）との効果比較**
- **個人特性・性格・ライフスタイルとの適合性**
- **ワークスタイル（リモート・オフィス等）別効果**
- **年齢・職業・環境による効果の差**
- **最適な適用場面・推奨条件**

---

### **📊 STEP 5: TEN評価軸詳細分析**

最終的に各項目のTEN評価を詳細根拠と共に確定：

#### **修正TEN評価軸（6項目）**
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

#### **各軸の詳細根拠収集（習慣特化）**
- **即効性**: 習慣開始から効果実感までの具体的期間・体感変化
- **簡単さ**: 実際の実践難易度・継続困難度・挫折率
- **人気度**: 実践者数・書籍・メディア露出・認知度
- **コスパ**: 時間投資対効果・必要コスト vs 生産性向上
- **機能専門性**: 科学的根拠の強さ・独自性・差別化要素
- **生産性UP度**: 実際の作業効率・集中力・パフォーマンス向上効果

## 🎯 品質チェックリスト

### 必須確認項目
- [ ] **科学的根拠**: 信頼できる研究・論文による裏付け確認
- [ ] **実践可能性**: 実際に継続実施可能な現実的な方法
- [ ] **効果測定**: 具体的・定量的な効果の確認
- [ ] **継続性評価**: 長期継続の困難さ・挫折要因の分析
- [ ] **個人差考慮**: 適用できる人・できない人の条件
- [ ] **コスト現実性**: 時間・金銭的投資の現実的評価
- [ ] **組み合わせ効果**: 他習慣との相乗効果・干渉リスク
- [ ] **適用場面**: 具体的な実践シーン・環境要件

### 絶対避けるべき項目
- [ ] **根拠薄弱**: 科学的根拠の薄い効果の誇張
- [ ] **継続困難**: 現実的に継続困難な方法の推奨
- [ ] **万能視**: 全ての人・場面に適用可能との誤解
- [ ] **即効期待**: 長期習慣を短期効果として誤解
- [ ] **コスト軽視**: 時間投資・機会費用の軽視
