## 📋 詳細リサーチ手順（5段階プロセス）

### **🔍 STEP 1: 製品仕様・可用性詳細収集**

各ガジェットについて以下の基本情報を詳細収集：

#### **検索クエリ例**
```
"[製品名] 2025 specifications features price review"
"[製品名] Japan availability purchase shipping support"
"[製品名] warranty customer service Japanese support"
"[製品名] compatibility requirements setup guide"
"[製品名] latest version updates features 2025"
```

#### **収集必須情報**
- **正式名称・製造会社・発売年・最新モデル**
- **技術仕様（サイズ・重量・バッテリー・接続方式等）**
- **価格（定価・実売価格・セール情報）**
- **日本国内販売状況（正規代理店・Amazon・サポート体制）**
- **保証制度・アフターサポート**
- **対応OS・デバイス・設定要件**

---

### **🎮 STEP 2: 実用性・生産性効果詳細調査**

#### **検索クエリ例**
```
"[製品名] productivity improvement real world usage"
"[製品名] user experience long term review 2025"
"[製品名] work from home setup benefits efficiency"
"[製品名] ergonomics health benefits workplace usage"
"[製品名] workflow integration daily usage tips"
```

#### **収集必須情報**
- **主要使用場面・ワークスタイル（5-7個の具体例）**
- **実際の生産性向上効果・時短効果**
- **長期使用での満足度・耐久性**
- **エルゴノミクス・健康への影響**
- **ワークフロー統合・他ガジェットとの連携**
- **日常使いでの実用性・利便性**

---

### **💰 STEP 3: コスト・投資効果詳細分析**

#### **検索クエリ例**
```
"[製品名] price comparison value for money ROI 2025"
"[製品名] cost effectiveness productivity gain analysis"
"[製品名] budget alternatives comparison similar products"
"[製品名] total cost ownership maintenance replacement"
```

#### **収集必須情報**
- **総所有コスト（本体・アクセサリ・メンテナンス・交換）**
- **価格対効果・投資回収期間**
- **予算別代替品・コスパ重視選択肢**
- **長期使用でのコストパフォーマンス**
- **アップグレード・買い替えサイクル**
- **中古市場・リセールバリュー**

---

### **⚔️ STEP 4: 競合比較・差別化分析**

#### **検索クエリ例**
```
"[製品名] vs [競合製品1] vs [競合製品2] 2025"
"[製品名] competitive advantages unique features"
"[製品名] market position comparison review ranking"
"[製品名] pros cons vs alternatives buying guide"
```

#### **収集必須情報**
- **直接競合製品（3-5個）とカテゴリ内位置**
- **独自機能・技術的差別化ポイント**
- **他製品に対する明確な優位性**
- **劣る点・課題・改善要望**
- **選択理由・推奨条件・適用場面**

---

### **📊 STEP 5: TEN評価軸詳細分析**

最終的に各項目のTEN評価を詳細根拠と共に確定：

#### **修正TEN評価軸（6項目）**
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

#### **各軸の詳細根拠収集（ガジェット特化）**
- **即効性**: 購入から生産性効果実感までの具体的期間・事例
- **簡単さ**: 実際のセットアップ・学習コスト・使いこなし期間
- **人気度**: 市場シェア・レビュー数・業界認知度
- **コスパ**: 価格対生産性効果・競合価格比較・投資回収
- **機能専門性**: 技術的優位性・独自機能・イノベーション度
- **生産性UP度**: 実際の作業効率・健康・集中力向上効果

## 🎯 品質チェックリスト

### 必須確認項目
- [ ] **実効果検証**: 実際の生産性向上効果の具体的測定
- [ ] **最新情報**: 2025年の最新モデル・価格・機能反映
- [ ] **日本購入**: 国内販売・サポート・保証の詳細確認
- [ ] **コスト分析**: 総所有コスト・投資効果の現実的評価
- [ ] **競合比較**: 同カテゴリでの客観的相対評価
- [ ] **長期使用**: 耐久性・満足度・買い替え時期の情報
- [ ] **導入容易性**: セットアップ・学習コストの正確な情報
- [ ] **適用場面**: 具体的な使用シーン・効果の詳細

### 絶対避けるべき項目
- [ ] **古い情報**: 2024年以前のモデル・価格での判断
- [ ] **机上の空論**: 実際の使用に基づかない理論的評価
- [ ] **効果誇張**: 現実的でない生産性向上効果の表現
- [ ] **コスト軽視**: 価格・維持費・機会費用の軽視
- [ ] **適用範囲**: 限定的な使用場面の過大評価
