**バージョン**: 1.0  
**対象**: Claude Code  
**目的**: AIツール上位項目の詳細情報収集・構造化データ作成

**出力**: JSON構造化データ + Instagram投稿用データ  
**基盤**: AIツールリサーチマスタープロンプトV3の後続プロセス

## 🎯 詳細リサーチ目的

**AIツール詳細情報の完全収集・構造化**
- **対象**: TENランキング上位5-10項目（一括処理）
- **目標**: Instagram投稿用の完全データセット作成
- **重点情報**: 
  - 実用性・使用感の詳細
  - 無料版 vs 有料版の具体的差
  - 日本語対応・サポート状況
  - 実際の生産性向上事例
  - 競合比較・差別化ポイント
  - 具体的な使用方法・コツ
- **品質基準**: 実際に試用可能なレベルの詳細情報

---

## 📋 詳細リサーチ手順（5段階プロセス）

### **🔍 STEP 1: 基本情報詳細収集**

各AIツールについて以下の基本情報を詳細収集：

#### **検索クエリ例**
```
"[ツール名] 2025 review detailed features pricing"
"[ツール名] vs alternatives comparison 2025"
"[ツール名] free vs paid features limitations"
"[ツール名] Japanese support UI language 2025"
"[ツール名] getting started tutorial guide"
```

#### **収集必須情報**
- **正式名称・開発会社**
- **リリース年・最新アップデート日**
- **価格体系（無料版・有料版詳細）**
- **日本語対応レベル（UI・サポート・ドキュメント）**
- **システム要件・対応環境**
- **ユーザー数・普及状況**

---

### **🎮 STEP 2: 実用性・使用感詳細調査**

#### **検索クエリ例**
```
"[ツール名] user experience review 2025 pros cons"
"[ツール名] real world use cases productivity"
"[ツール名] beginner tutorial how to use effectively"
"[ツール名] workflow integration productivity boost"
"[ツール名] learning curve getting started experience"
```

#### **収集必須情報**
- **主要用途・使用場面（3-5個）**
- **学習時間・習得難易度**
- **インターフェース・使いやすさ**
- **実際の生産性向上効果**
- **よくある使用方法・ベストプラクティス**
- **初心者向けの始め方・コツ**

---

### **💰 STEP 3: 料金・コスト詳細分析**

#### **検索クエリ例**
```
"[ツール名] pricing plans 2025 free tier limitations"
"[ツール名] cost effectiveness ROI productivity"
"[ツール名] free alternative comparison budget"
"[ツール名] subscription worth it value for money"
```

#### **収集必須情報**
- **無料版の具体的制限（回数・機能・品質等）**
- **有料版の価格・プラン詳細**
- **無料版での実用性・継続利用可能性**
- **コストパフォーマンス評価**
- **予算制約時の代替案**

---

### **⚔️ STEP 4: 競合比較・差別化分析**

#### **検索クエリ例**
```
"[ツール名] vs [競合ツール1] vs [競合ツール2] 2025"
"[ツール名] unique features competitive advantage"
"[ツール名] strengths weaknesses vs competitors"
"[ツール名] market position 2025 differentiation"
```

#### **収集必須情報**
- **主要競合ツール（2-3個）**
- **差別化ポイント・独自機能**
- **他ツールに対する優位性**
- **劣る点・課題・弱み**
- **選択理由・推奨条件**

---

### **📊 STEP 5: TEN評価軸詳細分析**

最終的に各項目のTEN評価を詳細根拠と共に確定：

#### **修正TEN評価軸（6項目）**
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

#### **各軸の詳細根拠収集**
- **即効性**: 導入から効果実感までの具体的時間・事例
- **簡単さ**: 実際の設定手順・学習コスト・UI評価
- **人気度**: ユーザー数・レビュー数・業界認知度
- **コスパ**: 価格対効果・無料版実用性・競合価格比較
- **機能専門性**: 技術的優位性・専門機能・イノベーション度
- **生産性UP度**: 実際の時短効果・効率化・作業品質向上

## 🎯 品質チェックリスト

### 必須確認項目
- [ ] **実用性検証**: 実際に試用可能なレベルの詳細情報
- [ ] **最新情報**: 2025年1月以降の最新状況反映
- [ ] **日本語対応**: UI・サポート・ドキュメントの実際の品質
- [ ] **無料版詳細**: 具体的な制限・実用性の正確な情報
- [ ] **競合比較**: 同カテゴリ内での客観的相対評価
- [ ] **生産性効果**: 具体的な時短・効率化効果の事例
- [ ] **初心者配慮**: 未経験者でも始められる情報・手順
- [ ] **コスト透明性**: 隠れコスト・追加料金の確認

### 絶対避けるべき項目
- [ ] **古い情報**: 2024年以前の情報での判断
- [ ] **机上の空論**: 実際に使用困難な理論的評価
- [ ] **過大広告**: 現実的でない効果の誇張
- [ ] **不正確な価格**: 料金体系・制限の間違った情報
- [ ] **競合無視**: 単独評価での過大・過小評価
