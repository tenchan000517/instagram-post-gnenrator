## 🎯 TEN評価軸（資格特化版・6項目評価）

### 算出式
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### 各項目評価基準（資格特化版）

#### **即効性 (20%) - キャリア効果の早さ**
- **100点**: 取得直後から転職・昇進・年収アップ効果
- **90点**: 3ヶ月以内にキャリア効果実感
- **80点**: 6ヶ月以内にキャリア効果実感
- **70点**: 1年以内にキャリア効果実感
- **60点**: 2年以内にキャリア効果実感
- **40点**: 長期的なキャリア効果
- **20点**: 効果が限定的・不明確

#### **簡単さ (20%) - 取得の難易度**
- **100点**: 1-2週間の学習で取得可能
- **90点**: 1-2ヶ月の学習で取得可能
- **80点**: 3-4ヶ月の学習で取得可能
- **70点**: 6ヶ月程度の学習が必要
- **60点**: 1年程度の学習が必要
- **40点**: 2-3年の長期学習が必要
- **20点**: 極めて困難・専門的知識必須

#### **人気度 (15%) - 企業・市場での認知度**
- **100点**: 業界標準・全世界で認知（TOEIC、AWS等）
- **90点**: 広く認知・多くの企業で評価（簿記、基本情報等）
- **80点**: 業界で高い認知度・転職で有利
- **70点**: 専門分野で高評価・特定業界で重視
- **60点**: 一定の認知度・履歴書記載価値
- **40点**: 新興・注目度上昇中
- **20点**: ニッチ・限定的認知

#### **コスパ (15%) - 費用対効果**
- **100点**: 完全無料・高い価値（Google認定等）
- **90点**: 低コスト・高い価値（基本情報、簿記等）
- **80点**: 標準的コスト・十分な価値
- **70点**: やや高コスト・相応の価値
- **60点**: 標準的な費用対効果
- **40点**: 高コスト・限定的価値
- **20点**: 極めて高コスト・価値に疑問

#### **機能専門性 (20%) - 専門性・希少価値**
- **100点**: 極めて高い専門性・希少価値（弁護士、医師等）
- **90点**: 高い専門性・明確な差別化（AWS上位資格等）
- **80点**: 専門性あり・業界で重視される
- **70点**: 一般的専門性・カテゴリ内で優秀
- **60点**: 基本的専門性・標準的価値
- **40点**: 汎用的・特別な専門性なし
- **20点**: 限定的専門性・代替多数

#### **生産性UP度 (10%) - キャリア・仕事効率への効果**
- **100点**: 圧倒的なキャリア向上・仕事効率大幅改善
- **90点**: 明確なキャリア効果・業務効率化
- **80点**: 一定のキャリア効果・スキル向上
- **70点**: 軽度のキャリア効果・専門性向上
- **60点**: 限定的なキャリア効果
- **40点**: キャリア効果が不明確
- **20点**: キャリア効果なし・取得負荷のみ
