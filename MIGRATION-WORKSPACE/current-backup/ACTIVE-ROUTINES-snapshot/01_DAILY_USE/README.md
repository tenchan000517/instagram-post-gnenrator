# 01_DAILY_USE - 毎日使用するファイル

このディレクトリには日常的に使用する重要なファイルが含まれています。

## 構成

### system-master/
- システム全体の制御と起動コマンド
- マスターターミナルと完全ロジックマニュアル

### feed-posts/
- フィード投稿生成の基本フロー
- START-PROMPT.mdが起点となる重要ファイル

### reel-master-prompts/
- リール投稿作成の最新マスタープロンプト
- 投稿生成・分析の統合プロンプト集

### quiz-generation/
- ストーリークイズ生成フロー
- 品質ガイドラインとマスターガイド

### habit-ranking/
- 習慣データベースとランキングシステム
- リサーチプロンプトとマスタープラン

## 使用方法

1. 新規作業開始時は `system-master/QUICK-START-COMMANDS.md` を確認
2. フィード投稿は `feed-posts/START-PROMPT.md` から開始
3. リール投稿は `reel-master-prompts/` の最新プロンプトを使用
4. クイズ作成は `quiz-generation/` のガイドラインに従う
