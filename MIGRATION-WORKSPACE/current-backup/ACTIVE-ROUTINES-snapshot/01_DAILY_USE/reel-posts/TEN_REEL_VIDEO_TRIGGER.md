# 🎬 TEN リール動画生成 起動術式

## 📋 実行コマンド

```
以下のマスタープロンプトを読み込んで、TEN版のリール動画台本を作成してください。

1. マスタープロンプトを読み込む:
/mnt/c/instagram-course/instagram-post-generator/ACTIVE-ROUTINES/04_REFERENCE/master-prompts/TYPE003生産性ランキング/REEL-VIDEO-MASTER-PROMPT-V2.md

2. 使用するKナレッジファイル:
/mnt/c/instagram-course/instagram-post-generator/app/data/knowledgeBase/knowledge/type003/K604.json
（または最新のK60x番台ファイル）

3. 作成する成果物:
- ナレーション用台本（カタカナ表記・5行構成）
- 投稿用テキスト（正式英語名称・3行構成）
- キャプション（マニュアル準拠）
- ハッシュタグ12個（効率化・生産性系）

【重要】ナレーションと投稿用の違い:
ナレーション：第X位＋ツール名（カタカナ）＋説明5行
投稿用：第X位（改行）＋ツール名（英語）＋説明3行
```

## 🎯 作成要件

### TEN版の特徴
- **構成**: 5行テンプレート（ツール名→一言説明→機能2行→効果2行）
- **口調**: 「〜なり」「〜であるぞ」等の古風な表現
- **フォーカス**: 効率化・生産性・時短効果を強調
- **冒頭**: 「これ知らないと損してます」を含む

### 出力フォーマット
```
## ナレーション用台本（読み上げ用）
例：
第5位：ジェンスパーク
最新の検索特化型AI
複数の視点から回答を自動生成
情報収集の精度が格段にアップ
調査時間を大幅に短縮可能

## 投稿用テキスト（画面表示用）
例：
第5位
Genspark
複数の視点から回答を自動生成
情報収集の精度が格段にアップ

## キャプション
[固定フォーマット準拠のキャプション]

## ハッシュタグ
[12個のハッシュタグ]
```

## 📝 確認事項

1. K60x番台のファイルからTOP5を抽出しているか
2. 60秒の時間配分（フック10秒・ランキング40秒・締め10秒）を守っているか
3. TENキャラクターの口調が一貫しているか
4. AIツール名はナレーションでカタカナ、投稿用で英語表記になっているか
5. キャプションのマニュアル準拠フォーマットを守っているか
