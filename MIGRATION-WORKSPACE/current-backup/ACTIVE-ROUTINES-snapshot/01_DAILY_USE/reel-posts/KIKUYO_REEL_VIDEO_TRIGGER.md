# 🎬 KIKUYO リール動画生成 起動術式

## 📋 実行コマンド

```
以下のマスタープロンプトを読み込んで、KIKUYO版のリール動画台本を作成してください。

1. マスタープロンプトを読み込む:
/mnt/c/instagram-course/instagram-post-generator/ACTIVE-ROUTINES/04_REFERENCE/master-prompts/TYPE003生産性ランキング/REEL-VIDEO-MASTER-PROMPT-V2.md

2. 使用するKナレッジファイル:
/mnt/c/instagram-course/instagram-post-generator/app/data/knowledgeBase/knowledge/type003/K805.json
（または最新のK80x番台ファイル）

3. 作成する成果物:
- ナレーション用台本（カタカナ表記・ランキング構成を読み上げ）
- 投稿用テキスト（正式英語名称・実際の表示用）
- キャプション（マニュアル準拠）
- ハッシュタグ12個（キャリア・転職系）

【重要】ナレーションと投稿用の違い:
ナレーション：読み上げ用（「第5位：メルカリ」のように口頭で伝える形式）
投稿用：画面表示用（第5位＋改行＋Mercari のように視覚的に見やすい形式）
```

## 🎯 作成要件

### KIKUYO版の特徴
- **構成**: 2-3行の簡潔構造（5位・4位・2位は2行、3位・1位は3行）
- **口調**: 「のです」は原則使用しない（自然な日本語優先）
- **フォーカス**: 企業情報・年収・キャリアを強調
- **冒頭**: 「これ知らないと損するのです」

### 出力フォーマット
```
## ナレーション用台本（読み上げ用）
例：
第5位：メルカリ
フリマアプリからフィンテックまで展開
多様な事業で急成長を続けている

## 投稿用テキスト（画面表示用）
例：
第5位
Mercari
フリマアプリからフィンテックまで展開
多様な事業で急成長を続けている

## キャプション
[固定フォーマット準拠のキャプション]

## ハッシュタグ
[12個のハッシュタグ]
```

## 📝 確認事項

1. K80x番台のファイルからTOP5を抽出しているか
2. 60秒の時間配分（フック10秒・ランキング40秒・締め10秒）を守っているか
3. 「のです」の使用を最小限に抑えているか（原則使用しない）
4. 企業名は投稿用で正式英語表記になっているか
5. 5位・4位・2位は2行、3位・1位は3行の構成になっているか
6. キャプションのマニュアル準拠フォーマットを守っているか
