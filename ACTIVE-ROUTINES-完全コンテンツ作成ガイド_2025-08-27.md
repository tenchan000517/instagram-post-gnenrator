# ACTIVE-ROUTINES 完全コンテンツ作成ガイド

**作成日**: 2025年8月27日  
**目的**: ACTIVE-ROUTINESディレクトリから作成可能な全コンテンツと手順の完全ドキュメント化  
**対象**: Instagram投稿（リール・フィード・ストーリーズ）のコンテンツ生成システム

---

## 📚 システム全体概要

ACTIVE-ROUTINESは以下5つのカテゴリで構成されています：

1. **01_DAILY_USE**: 日常使用ファイル（起動術式・マスタープロンプト）
2. **02_ACTIVE_CONTENTS**: アクティブコンテンツ（作成済み投稿）
3. **03_ANALYSIS_DATA**: 分析データ（キャラクター設定・バズ分析）
4. **04_REFERENCE**: リファレンス（マニュアル・ガイドライン）
5. **05_ARCHIVE**: アーカイブ（過去版・廃止版）

## 🎬 1. リール投稿（REEL CONTENT）

### ⚡ リール投稿の2つの作成方式

#### 1-1. マスタープロンプトベース（従来型）
**特徴**: ゼロからテーマ作成→5段階フロー→完成
**適用**: 全キャラクター対応（King・Misaki・Iida・Ten）

**起動ファイル**: `/01_DAILY_USE/reel-master-prompts/リール投稿作成_起動術式.md`

**起動コマンド**:
```bash
# King投稿作成例
kingの投稿を5個作成してください

# Misaki投稿作成例  
misakiの投稿を8個作成してください

# Iida投稿作成例
iidaの投稿を10個作成してください

# Ten投稿作成例
tenの投稿を7個作成してください
```

**詳細作成フロー（5段階）**
**マスタープロンプト**: `/04_REFERENCE/manuals/コンテンツ作成マニュアル/13_マスタープロンプト_2025-08-23.md`

**STEP1**: コア要素作成
- テーマ選定マニュアル参照
- ターゲット作成マニュアル参照

**STEP3**: キャプション・ハッシュタグ作成
**STEP4**: 品質チェック（58項目満点確認）
**STEP5**: 保存・INDEX更新
