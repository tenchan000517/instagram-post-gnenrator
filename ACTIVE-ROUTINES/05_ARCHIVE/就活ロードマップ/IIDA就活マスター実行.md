```

- あなたの性格に合う企業TOP10

- 性格タイプ別面接必勝法

【第2章 - 自己分析】

【第3章 - 企業分析】

【第4章 - 面接対策】
- 面接官の心を掴む7つの技術

【第5章 - 身だしなみ】
- 第一印象で勝つ身だしなみ
- 業界別ドレスコード完全版
- 小物で差をつける方法
```

### **STEP2: 10ページ構成設計**
```
【IIDA特別構成】
1ページ目: 章タイトル＋IIDAメッセージ
2ページ目: この章で得られる成果
3ページ目: よくある失敗パターン

5-8ページ目: 実践ステップ（4ページ）
9ページ目: チェックリスト/まとめ
10ページ目: 次のアクション・CTA
```

### **STEP3: IIDAキャラクター演出**
#### **IIDAの特徴**
```
【人物設定】
- 就活を終えたばかりの先輩
- 親身で温かいアドバイザー
- 実体験に基づく具体的助言
- 励まし・応援のメッセージ

【口調・トーン】
- 「〜だよ」「〜してみて」の親近感
- 「大丈夫！」「きっとできる」の励まし
- 「私も最初は〜」の共感
- 「一緒に頑張ろう」の伴走感
```

### **STEP4: 実践ステップ作成（5-8ページ）**

```

・[実践のコツ1]

✨ 成功例
「[具体的な成功事例や文例]」

⚠️ 注意点
[やってはいけないこと]
```

段落4: 実践ステップ（120-140文字）
