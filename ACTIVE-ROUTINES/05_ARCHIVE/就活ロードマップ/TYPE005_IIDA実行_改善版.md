## 🛠️ 実例中心生成プロセス（4ステップ）

### **STEP1: ロードマップから実例抽出**
- [ ] 該当章から具体的な事例・実例を特定
- [ ] 実際に使える数字・データ・企業名を抽出
- [ ] 内定者の実体験エピソードを優先選択
- [ ] テンプレート化できる部分を特定

### **STEP2: 実例のカテゴリ分け**
#### **使用可能な実例タイプ**
- ✅ 内定者の実際の回答全文
- ✅ 具体的な企業名・数字・データ
- ✅ そのまま使えるテンプレート
- ✅ 実際の面接での会話記録
- ✅ ES・履歴書の記載例

#### **排除する内容**
- ❌ 「〜のやり方」「〜の方法」
- ❌ 抽象的なアドバイス
- ❌ 理論的な説明
- ❌ 手順・ステップの説明

### **STEP3: IIDA実例投稿構成**
#### **導入部（実例予告）**
```
[具体的企業名]内定者が実際に使った[具体的内容]を全文公開します

これをそのまま使って、多くの就活生が結果を出しています
```

#### **メイン部分（実例提示）**
```
【実例】[タイトル]

[具体的な実例・全文・テンプレート]
※企業名、数字、固有名詞を含む実際の内容

【使い方】
あなたの経験に置き換えて、同じ構成で作成してください
```

#### **締めくくり（励まし）**
```
この実例を参考に、あなたの就活でも実践してみてください

必ず良い結果につながります
```

### **STEP4: 品質確認**
- [ ] 禁止語句が使用されていないか確認
- [ ] 実例・事例のみで構成されているか確認
- [ ] 学生が実際に使える内容か確認
- [ ] IIDAの励ましトーンが含まれているか確認

## 🎯 実例提供パターン

### **回答実例パターン**
```
【内定者実例】Google面接での自己紹介

「私は○○大学の○○と申します。大学では○○を学び、特に○○の研究で○○という成果を出しました。アルバイトでは○○で○年間働き、○○の改善により売上が○%向上しました。御社では○○の事業で、私の○○の経験を活かし、○○に貢献したいと考えています。」

※あなたの経験に置き換えてそのまま使用可能
```

### **テンプレート提供パターン**
```

4行目：将来のキャリアビジョン

※各行40-50字で構成
```

```
・平均ES文字数：387文字（400字以内設問）
・最頻出キーワード：「貢献」「成長」「挑戦」
・高評価エピソード：アルバイト（34%）、サークル（28%）、研究（23%）

※この データを参考に、あなたのESを調整してください
```

## 🎖️ 達成目標

**実例提供**: 内定者の実際の回答・テンプレート・データ  
**即効性**: コピー&ペーストで使える実用的内容  
**具体性**: 企業名・数字・固有名詞を含む具体的情報  
**差別化**: 他の就活生が知らない実際の成功例  
**励まし**: IIDA独特の温かい応援メッセージ

---

**重要**: この改善版では、方法論を一切排除し、実例・事例・テンプレートのみでコンテンツを構成します。学生が「どうやればいいか」ではなく「実際にはどうなのか」を知ることができる内容に特化しています。
