# ビジネスマナー系クイズ作成ガイドライン

## ビジネスマナージャンルの定義

**ビジネスマナー = 社会人基礎スキル**
- 基本的なビジネスマナー
- 電話・メール・会議のルール
- 名刺交換・挨拶・服装
- 社会人としての常識

## 作成フロー

### 1. ルール選定
ビジネスマナー書籍・研修資料から確立されたルールを選択

### 2. ルールの明確化
曖昧な表現を避け、明確な基準に翻訳

### 3. クイズ化
22文字以内の○×問題として構成

### 4. 解説作成
- わかりやすい解説
- マナー理由を（参考：）で補足

## 作成ルール

### 基本原則
1. **客観的正解**: 一般的に確立されたマナーに基づく明確な答え
2. **実用性**: 知らないと恥をかく・知ってると役立つ内容
3. **迷いポイント**: 80:20〜50:50の割合で迷いが生じること
4. **22文字制限**: 質問文は全角22文字以内

### 特徴
- 明確な正解があるルール系
- 知らないと恥をかく実用知識
- 客観的な判断基準

### 避けるべき内容
- TPO依存・ケースバイケース
- 数字あて（時間・角度等）
- 曖昧な表現（「適切」「無難」等）
- 明らかすぎる答え（常識レベル）

## 参考分野一覧

### 基本マナー
1. **挨拶・返事**
   - 時間帯別の挨拶
   - 相手に応じた敬語

2. **身だしなみ・服装**
   - ビジネススーツの基本
   - 清潔感の基準

3. **立ち居振る舞い**
   - 姿勢・歩き方
   - 座り方・立ち方

### コミュニケーション
4. **電話応対**
   - 受話器の持ち方
   - 話し方・聞き方

5. **メールマナー**
   - 件名・宛名の書き方
   - 返信のタイミング

6. **会議・打ち合わせ**
   - 座席の順序
   - 発言のタイミング

### 接客・営業
7. **名刺交換**
   - 渡し方・受け取り方
   - 保管方法

8. **来客応対**
   - 案内の仕方
   - お茶出しのマナー

## クイズ例（完成形）

```
✅【○×】名刺は両手で渡し片手で受け取る
答え：×
解説：名刺は両手で渡し、両手で受け取るのが基本です。相手への敬意を表します。
（参考：日本のビジネスマナー基本ルール）

✅【○×】電話に出るのは3コール以内が理想
答え：○
解説：3コール以内が基本です。それ以上だと相手を待たせてしまいます。
（参考：ビジネス電話応対マナー）

✅【○×】メールの返信は24時間以内が基本
答え：○
解説：ビジネスメールは24時間以内、できれば当日中の返信が基本マナーです。
（参考：ビジネスメールマナー）
```
