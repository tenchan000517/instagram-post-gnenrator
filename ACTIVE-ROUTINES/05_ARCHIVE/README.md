# 04_ARCHIVE - アーカイブ保管

過去のバージョンや古いファイルを安全に保管しています。

## 構成

### old-posts/
- 2025-08-15以前の古い投稿
- 旧バージョンのガイドライン・プロンプト
- 引き継ぎドキュメント類
- reel-archive/ (リール関連の古いファイル)

### old-indexes/
- 各キャラクターの古いINDEXファイル
- king_INDEX.md, misaki_INDEX.md, ten_INDEX.md, iida_INDEX.md
- 参照用として保管

### old-quiz/
- 古いクイズファイルとアーカイブ
- story-archive/ (ストーリークイズの過去版)
- 開発過程のクイズファイル

## 重要事項

⚠️ **削除厳禁**: このディレクトリのファイルは全て唯一のコピーです

## 保管ルール

- ファイルは削除せず、アーカイブに移動
- バージョン管理として旧版を保持
- 重要な引き継ぎ情報も保管
- 日付情報を保持して履歴追跡可能

## アクセス方法

- 通常の作業では参照不要
- 過去の経緯確認や緊急時の復元用
- 古いバージョンとの比較検討時に使用
