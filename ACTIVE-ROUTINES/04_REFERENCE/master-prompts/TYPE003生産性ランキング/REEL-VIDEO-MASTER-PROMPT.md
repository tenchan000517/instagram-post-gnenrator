## 📋 リール動画台本構造

### 1. フック部分（0-10秒）

#### 構成要素
```
【0-3秒】タイトル提示
【3-6秒】煽り・危機感演出  
【6-10秒】価値提供＋保存促進CTA
```

#### テンプレート
```
今すぐ導入すべきAIツールTOP5

これ知らないと損してます

特に1位は作業時間が3分の1になった
後で見返せるよう今すぐ保存して
```

#### 作成ルール
- **タイトル**: 「今すぐ導入すべき[ジャンル]TOP5」形式
- **煽り**: 「これ知らないと損してます」で統一
- **価値提供**: 1位の具体的効果を数値で表現
- **CTA**: 「保存して」で行動促進

---

### 2. ランキング部分（10-50秒）

#### 時間配分
```
各ランキング: 8秒×5項目 = 40秒
5位: 10-18秒
4位: 18-26秒  
3位: 26-34秒
2位: 34-42秒
1位: 42-50秒
```

#### ナレーション構造（5行テンプレート）
```
第X位：[ツール名]
[AIを一言で説明]
[機能説明1行目]
[機能説明2行目]
[効果説明1行目]
[効果説明2行目]
```

#### 各行の作成ルール

##### 1行目：ランキング表示
- 形式: 「第X位：[ツール名]」
- ツール名: 正式名称使用（Claude、ChatGPT、Perplexity等）

##### 2行目：一言説明
- 目的: AIの特徴を端的に表現
- 文字数: 10-15文字
- パターン例:
  - 「最新の検索特化型AI」
  - 「世界で最も使われているAI」
  - 「長文処理に最も優れたAI」
  - 「Google製の万能AI」

##### 3-4行目：機能説明
- 目的: 具体的な機能・特徴を2行で説明
- 各行文字数: 15-20文字
- 内容:
  - 3行目: 主要機能・処理能力
  - 4行目: 特徴的機能・差別化要素
- 例:
  ```
  複数の検索エンジンから情報を同時収集
  異なる視点から回答を自動生成
  ```

##### 5-6行目：効果説明
- 目的: 導入効果・メリットを2行で説明
- 各行文字数: 15-20文字
- 内容:
  - 5行目: 直接的効果・改善点
  - 6行目: 具体的な時短・効率化効果
- 例:
  ```
  情報収集の精度が格段にアップ
  調査時間を大幅に短縮可能
  ```

---

### 3. クロージング部分（50-60秒）

#### テンプレート

今日から1つでも導入してみて
