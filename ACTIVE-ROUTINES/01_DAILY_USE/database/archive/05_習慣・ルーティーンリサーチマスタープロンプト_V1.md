## 🎯 TEN評価軸（習慣特化版・6項目評価）

### 算出式
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### 各項目評価基準（習慣特化版）

#### **即効性 (20%) - 効果実感の早さ**
- **100点**: 初回実践から即座に効果実感（深呼吸、姿勢改善等）
- **90点**: 1週間以内に効果実感（早起き、読書習慣等）
- **80点**: 2週間以内に効果実感（運動習慣、瞑想等）
- **70点**: 1ヶ月以内に効果実感（学習習慣等）
- **60点**: 2-3ヶ月で効果実感（長期習慣等）
- **40点**: 半年以上で効果実感
- **20点**: 効果が不明確・個人差大

#### **簡単さ (20%) - 開始・継続の容易さ**
- **100点**: 思い立った瞬間に開始可能（感謝、深呼吸等）
- **90点**: 準備不要・即座に開始可能（散歩、水分補給等）
- **80点**: 最小限の準備で開始可能（読書、ストレッチ等）
- **70点**: 軽い準備・環境調整が必要
- **60点**: 道具・場所の確保が必要
- **40点**: 複数の条件・準備が必要
- **20点**: 高度な知識・技術が必要

#### **人気度 (15%) - 普及度・認知度**
- **100点**: 全世界で認知・多くの人が実践（運動、読書等）
- **90点**: 広く認知・多くの成功者が実践（早起き、瞑想等）
- **80点**: 生産性コミュニティで高い認知
- **70点**: 特定分野・業界で高評価
- **60点**: 一定の認知度・書籍・研究あり
- **40点**: 新しいアプローチ・注目上昇中
- **20点**: ニッチ・限定的認知

#### **コスパ (15%) - 費用対効果**
- **100点**: 完全無料・極めて高い効果（瞑想、感謝等）
- **90点**: 無料・高い効果（散歩、読書等）
- **80点**: 最小コスト・十分な効果
- **70点**: 低コスト・相応の効果
- **60点**: 標準的なコスト対効果
- **40点**: やや高コスト・限定的効果
- **20点**: 高コスト・効果に疑問

#### **機能専門性 (20%) - 科学的根拠・専門効果**
- **100点**: 多数の科学研究で実証・医学的根拠（睡眠、運動等）
- **90点**: 科学的根拠あり・専門家推奨（瞑想、間欠的断食等）
- **80点**: 一定の科学的根拠・研究結果あり
- **70点**: 経験的根拠・多くの成功事例
- **60点**: 基本的な理論・一般的効果
- **40点**: 理論的根拠・効果は個人差
- **20点**: 根拠薄い・効果不明確

#### **生産性UP度 (10%) - 実際の生産性向上効果**
- **100点**: 圧倒的な生産性向上・作業効率大幅改善
- **90点**: 明確な生産性向上・時短効果高い
- **80点**: 一定の生産性向上・効率化効果
- **70点**: 軽度の生産性向上・集中力向上
- **60点**: 限定的な生産性効果
- **40点**: 生産性向上効果が不明確
- **20点**: 生産性向上効果なし・時間コストのみ
