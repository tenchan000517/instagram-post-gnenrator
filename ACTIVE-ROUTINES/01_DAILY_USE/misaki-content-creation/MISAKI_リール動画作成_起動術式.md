**対象**: MISAKIキャラクター・女性向けコンテンツ  
**目的**: 10ページ構成リール動画の体系的作成  
**基準**: コンテンツ1品質レベルの維持

---

```
Claude、以下の手順でMISAKIリール動画を作成してください：

1. コンテンツ番号を指定してください（例：コンテンツ2）
2. 元コンテンツとタイトル4タイプを確認します
3. 10ページ構成で作成し、キャプションも生成します
4. 最終ファイルをmisaki_content_XXX.mdで保存します
```

## 🛠️ 作成手順（5ステップ）

### STEP1: 基盤情報収集
```
□ 指定されたコンテンツ番号の元コンテンツを読み取り
□ 対応する4タイプタイトルを確認
□ 元コンテンツの核心メッセージを理解
```

### STEP2: 比較表作成（3ページ目）
```
□ -タイプと÷タイプから8選ずつ抽出
□ 12文字以内でタイトル作成（例：「楽な道を選ぶ人」）
□ 各項目12文字以内で調整
□ 女性らしいあるある感を確保
□ 正確な対比構造を維持
```

### STEP3: コツページ作成（5-9ページ目）
```
□ +タイトルの語尾を正確に把握（「〜したこと」「〜した行動」「〜した方法」）
□ 語尾に応じて5-9ページの内容を調整：
  ・「〜したこと」→ 結果・体験談形式
  ・「〜した行動」→ 具体的な行動・アクション形式  
  ・「〜した方法」→ 手法・やり方形式
□ 元コンテンツと×タイプの掛け算を実現
□ 仕事での挑戦に統一（一貫性確保）
□ 意図的・能動的なエピソードに調整
□ MISAKIらしい温かい表現で統一
□ 改行・文字数を統一（コピペしやすく）
```

### STEP4: 全体構成確認
```
□ 10ページ構成の完成
□ 元コンテンツとの繋がり確認
□ MISAKIキャラクター一貫性チェック
□ リアリティ・共感性の確認
```

### STEP5: キャプション・保存
```
□ キャプションマニュアルに従って作成
□ MISAKI用掴み文（共感寄り添い型）を選択
□ 女性向けハッシュタグセットを適用
□ コピペしやすいフォーマットに調整：
  ・**太字**を削除
  ・1. 2. 3.の数字を削除
  ・「左:」「右:」を削除
□ ファイルを指定形式で保存
```

## 🔧 トラブルシューティング

### よくある失敗と対処法

**問題**: +タイトルと5-9ページの内容が不一致
**対処**: 「〜したこと」「〜した行動」「〜した方法」の語尾を正確に把握し、内容を合わせる

**問題**: 元コンテンツとの繋がりが薄い
**対処**: 元コンテンツの核心「困難な環境に身を置く」を5つのエピソード全てに反映

**問題**: 受動的なエピソードになる
**対処**: 「意図的に選んだ」「自分から挑戦した」表現に変更

**問題**: エピソードがワンパターン
**対処**: 人間関係・スキル・評価・視野・楽しさの5つの観点で差別化

**問題**: 文才を履き違える
**対処**: 小説調・ポエム調は避け、SNS向けシンプル文章に
