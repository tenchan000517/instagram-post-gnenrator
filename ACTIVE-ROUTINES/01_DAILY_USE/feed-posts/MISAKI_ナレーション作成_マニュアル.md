# MISAKI ナレーション作成マニュアル

**対象**: MISAKIリール動画用ナレーション  
**目的**: 親しみやすく共感を呼ぶナレーション作成  
**基準**: 002クオリティ準拠・530文字程度

---

## 📊 基本仕様

### **文字数仕様**
- **総文字数**: 500-550文字（目安530文字）
- **段落構成**: 6段落
- **文の長さ**: 短文と中文のバランス（15-30文字/文）

### **語り口調仕様**
- **基本トーン**: 親しみやすい関西弁風
- **語尾パターン**: 「〜ですよね」「〜なんです」「〜だったけど」
- **ひらがな比率**: 高め（60-70%）
- **読点使用**: 多用（自然な会話調）

---

## 🎯 構造テンプレート（最適化版・簡潔構成）

### **段落1: 導入（40-60文字）**
```
[元コンテンツの核心]って思う時、[重要なポイント]ですよね。[効果的なアプローチ]なんです。
```
**例**: 夢や目標を実現したいって思う時、まずは明確な目標設定が大切ですよね。でも実現するための一番のポイントって、実は環境と習慣を変えることなんです。

### **段落2: 効果説明（20-30文字）**
```
[良い習慣の効果]。
```
**例**: 良い習慣が心も体も人間関係も良くしてくれる。

### **段落3: 対比説明（60-80文字）**
```
[できない人の特徴]になりがち。一方で[できる人の特徴]。
```
**例**: 目標が達成できない人は曖昧な目標設定で口先だけの宣言になりがち。一方で目標達成した人は明確な目標を設定して、計画をしっかり実行に移していく。

### **段落4: タイトル提示（1回のみ）**
```
[+タイトル]。
```
**例**: 目標を叶えた人がやってた5つの習慣。

### **段落5: 5つの具体例（各30-40文字・超簡潔）**
```
まず[習慣1簡潔]。[不安表現]けど、[結果]。

次に[習慣2簡潔]。[状況]時も、[効果]んです。

[数字][習慣3簡潔]。[行動理由]ことで[結果]。

[数字][習慣4簡潔]。[期間]けど[結果]に。

さいごは[習慣5簡潔]。[最初の印象]けど[継続効果]。
```

### **段落6: CTA（30-40文字）**
```
参考になったと思ってもらえたらいいねやコメントで教えてもらえるとうれしいです！
```

## 🆚 構造比較（冗長版 vs 最適版）

### **❌ 冗長構造（旧版）**
- タイトル2回提示
- 各習慣詳細説明4行
- 総文字数550-600文字
- 接続詞・修飾語多用

### **✅ 最適構造（新版）**  
- タイトル1回のみ
- 各習慣1-2文に簡潔化
- 総文字数530文字以内
- 核心のみ・直接的表現

## 🎨 MISAKIらしい表現集

### **導入・共感フレーズ**
```
✅ 「〜ですよね」「〜ばかりなんです」「〜なことって多いですよね」
✅ 「実は〜の方が」「意外と〜だったりします」
✅ 「〜って感じる人、多いと思います」
```

### **不安・心配の表現**
```
✅ 「『受からないかも』って不安だったけど」
✅ 「怒られるかもって心配だったけど」
✅ 「『今更始めても』って躊躇してたけど」
✅ 「『まだ準備不足かも』って思ったけど」
```

### **行動理由の表現**
```
✅ 「やらない後悔の方が大きいと思って」
✅ 「まずは話を聞くだけでもと思って」
✅ 「体験だけでもと思って」
✅ 「完璧じゃなくても始めてみようと思って」
```

### **結果表現**
```
✅ 「意外と楽しくて」「真剣に聞いてもらえて」
✅ 「新しいスキルが身についてきました」
✅ 「新しい自分を発見できたんです」
✅ 「自分の可能性を改めて知ることができて」
```

## 📝 ひらがな使用ルール

### **積極的にひらがな化する漢字**
```
そう → そう
こと → こと
とき → とき
って → って
けど → けど
から → から
でも → でも
まで → まで
```

### **カタカナ・漢字のまま残す語**
```
スキル、プロジェクト、エージェント
資格、上司、転職、副業、習い事
```

## 🔧 作成手順

### **STEP1: 基本情報整理**
1. +タイトルを確認
2. 元コンテンツの核心メッセージを把握
3. 5つの習慣内容を整理

### **STEP2: 段落別下書き**
1. 段落2: 元コンテンツを親しみやすく要約
2. 段落3: 対比を分かりやすく説明
3. 段落5: 5つの習慣を体験談風に変換

### **STEP3: MISAKIらしさ調整**
1. 硬い表現を柔らかく変換
2. ひらがな比率を調整
3. 読点を適切に配置

### **STEP4: 文字数調整**
1. 総文字数を530文字前後に調整
2. 各段落のバランスを確認
3. 読み上げやすさをチェック

## ⚡ 品質チェックリスト

### **構造チェック**
- [ ] 6段落構成になっているか
- [ ] タイトルが2回提示されているか
- [ ] 5つの習慣が順番通りか

### **語り口調チェック**
- [ ] 親しみやすい関西弁風になっているか
- [ ] 「〜ですよね」「〜なんです」が適切に使用されているか
- [ ] 硬い表現が残っていないか

### **文字数チェック**
- [ ] 総文字数が500-550文字の範囲内か
- [ ] 各段落のバランスが適切か
- [ ] 文が長すぎないか（30文字以内）

### **ひらがな比率チェック**
- [ ] ひらがな比率が60-70%程度か
- [ ] 読みやすいひらがな化ができているか
- [ ] 重要な漢字は残されているか

## 🚨 禁止表現・注意事項

### **禁止表現**
```
❌ 「戦略的に」「効率的に」「最適化」
❌ 「ステップ1」「①②③」等の機械的表現
❌ 「〜である」「〜だ」等の断定口調
❌ 「〜すべき」「〜しなければならない」
```

### **注意事項**
- 体験談風でも「私は」「私が」は使わない
- 具体的すぎる固有名詞は避ける
- 読み上げた時のリズムを意識
