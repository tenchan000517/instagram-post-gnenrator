## 🎯 10ページ構成（全キャラクター共通）

### 基本構造
```
1ページ目: メインタイトル（+タイプ）
2ページ目: 元コンテンツそのまま掲載
3ページ目: 比較表（-タイプ vs ÷タイプ）
4ページ目: メインタイトル再掲
5-9ページ目: +タイトルに応じた内容5つ
10ページ目: CTA（固定・作成不要）
```

### 重要な法則
```
+タイプの語尾で5-9ページの内容形式が決まる:
「〜したこと」→ 結果・体験談形式
「〜した行動」→ 具体的なアクション形式  
「〜した方法」→ 手法・やり方形式
```

## 🔧 作成手順（5ステップ）

### STEP1: 基盤情報収集
```
□ 指定されたコンテンツ番号の元コンテンツを読み取り
□ 対応する4タイプタイトルを確認
□ 元コンテンツの核心メッセージを理解
□ +タイプの語尾を正確に把握（「こと」「行動」「方法」）
```

### STEP2: 比較表作成（3ページ目）
```
□ -タイプと÷タイプから8選ずつ抽出
□ 12文字以内でタイトル作成
□ 各項目12文字以内で調整
□ キャラクター特性に応じたあるある感を確保
□ 正確な対比構造を維持
□ コピペしやすいフォーマット（太字・数字・左右表記削除）
```

### STEP3: コツページ作成（5-9ページ目）
```
□ +タイトルの語尾に応じて内容形式を調整
□ 元コンテンツと×タイプの掛け算を実現
□ 統一テーマで一貫性確保
□ 意図的・能動的なエピソードに調整
□ キャラクターらしい表現で統一
□ 改行・文字数を統一（コピペしやすく）
```

### STEP4: 全体構成確認
```
□ 10ページ構成の完成
□ 元コンテンツとの繋がり確認
□ キャラクター一貫性チェック
□ リアリティ・共感性の確認
```

### STEP5: キャプション・保存
```
□ キャプションマニュアルに従って作成
□ キャラクター用掴み文を選択
□ ターゲット向けハッシュタグセット適用
□ コピペしやすいフォーマットに調整
□ ファイルを指定形式で保存
```

## 🚨 よくある失敗パターンと対処法

### 致命的な失敗
```
問題: +タイトルと5-9ページの内容が不一致
例: 「〜した行動」なのに「〜したこと」の内容
対処: 語尾を正確に把握し、形式を合わせる
```

```
問題: 元コンテンツとの繋がりが薄い
例: 表面的な体験談のみで深い学びがない
対処: 元コンテンツの核心メッセージを5つ全てに反映
```

```
問題: 受動的なエピソード
例: 「仕方なく」「逃げ場がなくて」
対処: 「意図的に選んだ」「自分から挑戦した」に変更
```

### 品質の問題
```
問題: エピソードのバリエーション不足
対処: 統一テーマ内で5つの異なる観点を用意

問題: 文才の履き違え
対処: 小説調・ポエム調は避け、SNS向けシンプル文章

問題: キャラクター無視の表現
対処: 各キャラクターの表現特性を厳守
```
