# ESSENCE Method クイックガイド

作成日: 2025-08-26  
**実務用簡易版**

## 基本プロセス（7ステップ）

### 1. Extract（抽出）- コンテンツの核心
```
Q: このコンテンツで視聴者に一番伝えたいことは？
→ 1行で答える
```

### 2. Specify（特定）- 視聴者の変化  
```
現状: 視聴者の今の状態は？
理想: どうなってほしい？
```

### 3. Select（選択）- アプローチ選択
```
①体験談型：キャラ実体験で共感
②比較対照型：Before/After比較  
③問題解決型：具体的解決策
④気づき型：新しい視点提示
```

### 4. Express（表現）- キャラクターらしく

#### MISAKI
```
導入：「本音を言います」「実は私も」
展開：体験談＋共感＋希望
結び：「昨日の自分より」「ほんの少しだけ」
```

#### IIDA  
```
導入：「正直に言います」「君の気持ちわかる」
展開：分析＋勇気づけ＋方法
結び：「君なら必ずできる」「一緒に頑張ろう」
```

#### KING
```
導入：「俺の経験では」「本質はこれだ」
展開：分析＋断定＋筋トレ哲学
結び：「必ず結果が出る」「やるかやらないか」
```

### 5. Name（命名）- 魅力的タイトル

#### キャラ別パターン
```
MISAKI：「私が〜した結果」「〜って思える話」
IIDA：「〜してわかったこと」「君も〜してない？」  
KING：「〜の真実」「なぜ〜なのか」
```

#### 4つの基準
- 共感性：見たいと思うか
- 具体性：内容がわかるか  
- 期待感：価値が見えるか
- 一貫性：キャラらしいか

### 6. Check（確認）- 品質チェック

#### 必須チェック項目
```
□ コンテンツの核心が表現できてる？
□ 視聴者の変化に導く内容？
□ キャラクターらしい語り口？
□ 読後に前向きになれる？
□ タイトルと内容が一致？
```

### 7. Execute（実行）- 投稿完成
```
□ 適切な文章量
□ 読みやすい改行
□ 自然なCTA（行動促進）
```

## 緊急時5分版

### 超簡単3ステップ
```
1. コンテンツ3回読み→核心を1行でまとめ
2. キャラの定型パターンで執筆
3. タイトル3案作成→最良選択
```

### 迷った時の判断基準
```
1. 視聴者の価値（読んで良かった？）
2. コンテンツ本質（元メッセージから外れてない？）
3. キャラらしさ（その人が言いそう？）
```

## 失敗しがちなポイント

### ❌ やってはいけないこと
```
・フレームワーク埋めを目的にする
・リスト化を無理やりする  
・コンテンツから逸脱する
・キャラの体験を創作する
・抽象的なタイトル
```

### ✅ 成功のコツ
```
・視聴者のゴールから逆算
・本質メッセージを最優先
・キャラの体験談を活用
・自然な語り口を意識
・具体的で魅力的なタイトル
```

## キャラ別禁止表現

### MISAKI
```
❌「甘えている」「完璧にこなせば」「〜ませんか」「〜ね」
✅「無理しなくて」「あなたのペースで」「〜してください」
```

### IIDA
```
❌上から目線、押し付けがましい表現
✅「君なら」「一緒に」「必ずできる」
```

### KING  
```
❌感情軽視、論理だけ
✅「筋トレと同じ」「必ず結果が出る」「本質は」
```
