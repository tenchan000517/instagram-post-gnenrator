# KING リール動画作成マニュアル

**作成日**: 2025-08-27  
**目的**: KINGコンテンツのリール動画作成プロセスの体系化

---

## 📋 基本構成（10ページ固定）

1. **1ページ目**: メインタイトル（+タイプ）
2. **2ページ目**: 元コンテンツそのまま掲載
3. **3ページ目**: 比較表（-タイプvs÷タイプ）
4. **4ページ目**: メインタイトル再掲
5. **5-9ページ目**: ダメだった習慣5つ（改善前→改善後）
6. **10ページ目**: CTA（固定・作成不要）

---

## 🎯 コンテンツ連動構造

### ファイル連携
- **元コンテンツ**: `社会人向けコンテンツ100個02.md`（激励メッセージ）
- **タイトル集**: `コンテンツタイトル02.md`（4タイプのタイトル）

### 各タイプの役割
- **+タイプ**: メインタイトル（5選固定・ネガティブ系）
- **-タイプ**: ネガティブ比較（8選固定）  
- **×タイプ**: 実践コツの裏テーマ（5選固定）
- **÷タイプ**: ポジティブ比較（8選固定）

---

## ✅ 3ページ目：比較表作成ルール

### 必須要件
- 比較対象タイトル12文字以内（例：成長できない人 / 急成長する人）
- 各項目12文字以内
- 「〇〇な人の特徴8選」で統一
- 左右で正確な対比構造
- **番号なし（1.2.3.を使用しない）**

### 品質基準
- リアリティのある職場あるある
- 男性社会人特有の悩みを反映
- 攻めの表現で本音を突く

### 完成例フォーマット
```
成長できない人
急成長する人

成長できない人の特徴8選

「でも」が口癖

言い訳を先に考える

現状維持を好む

...

急成長する人の特徴8選

「まずやってみる」が基本

結果から学ぶ姿勢

変化を歓迎する

...
```

## 🚀 5-9ページ目：改善ページ作成の極意

### KING流の構成
**ダメだった習慣 → 改善した結果 = 成長**

- 表面：具体的なダメ習慣の改善
- 深層：自分に負荷をかけて成長する方法

### 元コンテンツとの繋がり（必須）
1. **意図的に**困難に挑戦する（能動的）
2. **自分から**負荷をかける決断
3. **早い段階で**行動に移す
4. 結果として圧倒的な成長を得る

### 使うべき表現
- 攻めの表現（「ダメだった」「致命的だった」）
- 熱血系の激励（「君ならできる」「俺が保証する」）
- 具体的な行動指示
- ビジネス現場のリアル

### エピソード作成のコツ
1. **リアリティ**: 職場でよくある失敗
2. **一貫性**: テーマを統一（例：仕事での成長）
3. **具体性**: 数字や期限を含める
4. **共感性**: 20-30代男性社会人の実情

### 文体の注意点
- 断定的で力強い文章
- 短文でインパクト重視
- 「残念な」「ダメな」を効果的に使用
- KINGらしい熱血表現

### コピペ対応の重要ポイント
- **リール動画構成部分は装飾記号を一切使用しない**
- **（太字）、左：右：などの記号は削除
- 対比の1. 2. などの番号付きリストも削除
- コピーしてそのまま投稿に使える形式を徹底

## 📝 KINGキャラクター特性

### 使う表現
- 攻めの言葉（「残念な」「ダメな」「致命的な」）
- 熱血系の激励（「君ならできる」「絶対に変われる」）
- 「俺が保証する」「教えてやる」
- ビジネス用語・成果重視の表現

### 使わない表現
- 優しすぎる表現
- 曖昧な言い回し
- 女性的な感情表現
- 弱気な表現

---

## 🔄 作成プロセス

1. **元コンテンツ確認**: 激励メッセージの核心を理解
2. **タイトル確認**: 4タイプの役割と内容を把握
3. **比較表作成**: 職場あるあるの対比を作成
4. **改善ページ作成**: ダメ習慣→改善結果を5つ作成
5. **全体調整**: 改行・文字数・表現の統一

## 📌 改善ポイント

### よくある失敗
- 元コンテンツとの繋がりが薄い
- 優しすぎる表現
- エピソードが抽象的
- 数字や具体性の欠如
- キャラクター無視の表現

### 成功のコツ
- 元コンテンツの激励を深く理解する
- 「俺もそうだった」と共感できる内容
- 具体的な改善アクションを提示
- KINGの熱血キャラクターを貫く
