## 📝 使用方法

### **新投稿作成時**
1. 該当キャラクターのファイルを確認
2. ターゲット層に合ったコンテンツを選択
3. そのメッセージを1ページ目のベースとして活用
4. 従来のフック+ターゲット+INDEXシステムと組み合わせ

### **分析・研究時**
- `extracted_text.md`: word01由来の元データで傾向分析
- `extracted_text02.md`: word02由来の追加データ（作成予定）
- `find-to-do01.md/02.md`: バズ要素の研究
- キャラクター別ファイル: 特定層向け最適化研究

### **ソースファイル管理**

- **word02**: `/archive/contents/word02/` - 分析待ち、extracted_text02.mdとして保存予定（100枚制限）
- **word03**: `/archive/contents/word03/` - word02から移動した101枚目以降（約30枚）

## 🔧 word02テキスト抽出手順

### **段階的抽出プロセス**
画像が多数あるため、一度に全てを処理せず段階的に抽出・追記を行います。

#### **手順1: 初回抽出**
1. word02の最初の10-20枚の画像をReadツールで読み取り
2. 抽出したテキストを`extracted_text02.md`に記載
3. プログレス確認

#### **手順2: 段階的追記**
1. 次の10-20枚の画像を読み取り
2. `extracted_text02.md`に**追記**（上書きではない）
3. プログレス確認
4. 完了まで繰り返し

#### **手順3: 完了確認**
- 100枚全ての処理完了確認
- extracted_text02.mdの内容品質チェック
- README更新（完了ステータス）

### **注意事項**
- **追記方式**: 既存内容を上書きしない
- **プログレス管理**: TodoWriteツールで進捗追跡
- **品質保持**: 各段階で内容確認
