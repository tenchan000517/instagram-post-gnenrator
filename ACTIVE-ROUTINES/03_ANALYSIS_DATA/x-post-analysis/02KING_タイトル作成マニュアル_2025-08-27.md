# KINGタイトル作成マニュアル

作成日: 2025-08-27  
対象: KINGキャラクター専用（20-40代男性社会人向け）  
範囲: コンテンツ読み込み～タイトル確定（フロー1-7）
ベース: MISAKIマニュアルのKING特化版

---

## フロー概要

1. **コンテンツ読み込み**
2. **成功定義**（コンテンツから「成功＝何か」を考える）
3. **事例参照**（Sランク・Aランク事例集を読み込む）
4. **タイトル作成**（+-×÷の4つ作成）
5. **自己評価**（頻出チェック・事例比較チェック）
6. **修正**（必要に応じて）
7. **出力**（タイトル4つ確定）

---

## STEP 1: コンテンツ読み込み

対象コンテンツファイル（社会人向けコンテンツ100個02.md）から指定されたコンテンツを3回熟読する。

**注意**: ファイルが複数存在する可能性があるため、どのファイルのどのコンテンツかを明確にする。

---

## STEP 2: 成功定義

**質問**: このコンテンツで視聴者に「どうなってほしいか」？

**KING向け成功定義の特徴**:
- 具体的な成果・結果を重視
- 行動力・実行力の向上
- キャリア・スキル・成長の実現
- 自分を変えるマインドセット構築

**例**:
- コンテンツ1（困難な環境で成長）→ 挑戦して成果を出す勇気を持つ
- コンテンツ4（目の前の仕事に集中）→ 集中力を高めて疲れを減らす

## STEP 4: タイトル作成（KING特化版）

### +-×÷の4タイプをKING向けに作成

**+タイプ**: ゴールに向かう方法・行動・習慣（必ず5選固定）
- KING特化: 「人生を好転させる習慣5選」
- 「○○を実現するためにしたこと5選」
- 「仕事ができる人の行動5選」
- 「生産性を上げる方法5選」
- **重要**: 内容が「行動」「方法」「こと」「習慣」かを明確にする（×タイプとリンク必須）

**-タイプ**: ゴールに到達できない人（比較対象用・÷タイプと対比）
- KING特化: 「結果が出ない人」
- 「成長できない人」
- 「チャンスを逃す人」
- **重要**: 「○○な人」「○○できない人」の形式で、÷タイプと必ず対比関係にする

**×タイプ**: ゴールのための具体的メソッド・手法（+タイトルとリンク）
- KING特化: 「○○を実現する方法」（+が「方法」の場合）
- 「○○するための行動」（+が「行動」の場合）
- 「○○を身につけるコツ」（+が「こと」や「習慣」の場合）
- **重要**: +タイトルの内容と必ずリンクする（行動→行動、方法→方法）

**÷タイプ**: ゴールに既に到達した人（比較対象用・-タイプと対比）
- KING特化: 「結果を出す人」
- 「成長し続ける人」
- 「チャンスを掴む人」
- **重要**: 「○○な人」「○○できる人」の形式で、-タイプと必ず対比関係にする

### 数字の範囲（リール動画投稿用カスタマイズ）
- **+タイプ**: 必ず5選で固定
- **×タイプ**: +タイトルとリンク・具体的手法（数字表記なし）
- **-タイプ**: 比較対象用（÷タイプと対比関係）
- **÷タイプ**: 比較対象用（-タイプと対比関係）

## STEP 5: 自己評価（KING特化チェック）

### 頻出チェック（マスト）
- [ ] 同じ表現を使いすぎていないか
- [ ] 前のコンテンツで使った表現と被っていないか
- [ ] KINGらしい熱血・激励表現になっているか
- [ ] **数字固定チェック**: +は5選、×は5つ、-は8選、÷は8選になっているか

### KINGキャラクター適合チェック
- [ ] **熱血度チェック**: 「君なら絶対できる！」感があるか
- [ ] **結果重視チェック**: 具体的な成果・変化を示しているか
- [ ] **行動促進チェック**: 「やってみろ！」という行動を促す表現か
- [ ] **男性向けチェック**: ビジネス・キャリア・成長感があるか

### 事例比較チェック
- [ ] Sランク事例と比べて突飛になっていないか
- [ ] ワードセンスが自然か
- [ ] 男性社会人が日常的に使う表現になっているか

### 📋 高品質チェック（KING版）

#### レベル1：基本ワードセンス
- [ ] **二重表現チェック**：「○○作り○つの方法」等の重複表現なし
- [ ] **硬すぎチェック**：「術」「システム」等が多すぎない
- [ ] **KING調チェック**：熱血・激励感がある自然な表現

#### レベル2：表現バリエーション（KING特化）
- [ ] **成果表現活用**：「結果を出す」「成功する」「達成する」でバリエーション
- [ ] **行動表現活用**：「やってみろ」「挑戦しろ」「実行しろ」で行動促進
- [ ] **成長表現活用**：「変わる」「成長する」「レベルアップする」

#### レベル3：コンテンツ理解（KING版）
- [ ] **本質理解チェック**：コンテンツの真意を男性向けに正確に反映
- [ ] **文脈適合チェック**：社会人男性の課題・関心に合致
- [ ] **KINGらしさチェック**：熱血で可能性を信じる表現

#### レベル4：高品質化（KING版）
- [ ] **具体性強化**：「年収○○万円」「○年で」等の具体的数字・期間
- [ ] **体験重視**：「実際にやってみた」「結果が出た瞬間」で臨場感
- [ ] **男性共感重視**：理論より実践・結果重視
- [ ] **激励最優先**：「君ならできる」感を最重要視

## KINGキャラクター専用注意事項

### 必須の表現特性
- **熱血激励**: 「君なら絶対できる！」「この方法で自分を変えてみろ！」
- **結果重視**: 具体的な成果・変化・数字を重視
- **行動促進**: 「やってみろ」「挑戦しろ」「実行しろ」
- **可能性信頼**: 「君の可能性はこんなもんじゃない」

### 禁止表現
- 優しすぎる表現（「〜ですね」「〜かもしれません」）
- 受動的な表現（「〜された」「〜してもらう」）
- 弱気な表現（「〜かな」「〜だと思う」）

### ターゲット専用キーワード
- ビジネス・キャリア・年収・出世・スキルアップ
- 成長・変化・結果・成果・達成・実現
- 行動・実行・挑戦・継続・積み重ね
- 可能性・能力・強さ・レベルアップ

## 🎯 弱小アカウント向け重要ポイント

### 必須理解事項
- **強アカウントとの違い**: 我々は弱小なので「誰のこと」が一瞬でわからないタイトルは使えない
- **本質を押さえる**: 読み手が「成長したい」「行動したい」と思うことが成功
- **元コンテンツに引きずられない**: 本質さえつかめば、元の言葉に縛られる必要なし

### 効果的なタイトルパターン

#### 読み手特定型（推奨）
```
「○○できない人の特徴5選」
「○○しない人の原因5選」
「○○な人がやりがちなこと5選」
```

#### KING特有の攻め表現（適度に使用）
```
「○○な人の残念な特徴5選」
「○○できない人のダメな習慣5選」
「○○しない人の致命的な原因5選」
```
※「残念な」「ダメな」「致命的な」など、刺激的だが品のある表現でKINGらしさを出す

#### 読み手欲求直接型
```
「○○する方法5選」
「○○のコツ5選」
「○○するための習慣5選」
```

### NG表現（絶対避ける）
- 「ヤバい」などの安直表現
- 「筋トレ」などジャンル判定危険ワード
- 「人の」「やってること」など他人事表現
- 「瞬間」「証拠」「理由」など不自然な言葉
- 誰のことかわからない内心告白系
