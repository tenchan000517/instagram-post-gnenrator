# KING タイトル作成起動術式

作成日: 2025-08-27  
対象: KINGキャラクター専用タイトル作成  
レベル: 次世代Claude Code用完全版

---

## 🚀 起動コマンド

```
Claude、KINGタイトル作成マニュアルに従って、
社会人向けコンテンツ100個02のタイトル集を作成してください。

【作業フロー】
1. コンテンツを読み込み（3回熟読）
2. 成功定義を明確化（読み手が「成長したい」「行動したい」と思うこと）
3. 弱小アカウント用タイトル作成（「誰のこと」が一瞬でわかる）
4. 4タイプ作成（+5選、-÷対比、×リンク）
5. 自己評価とKINGらしさチェック
```

---

## ⚡ 高速作成チェックリスト

### STEP1: コンテンツ理解
- [ ] コンテンツの本質を一言で把握
- [ ] 読み手に何を感じてほしいか明確化
- [ ] 元コンテンツの言葉に引きずられない

### STEP2: タイトル作成
- [ ] **+タイプ（必ず5選）**: 読み手特定型 or 欲求直接型
- [ ] **-タイプ（対比用）**: できない人・しない人
- [ ] **÷タイプ（対比用）**: できる人・する人（-と必ず対比）
- [ ] **×タイプ（リンク）**: +の内容に合わせた具体的手法

### STEP3: KINGらしさチェック
- [ ] 適度な攻め表現（「残念な」「ダメな」「致命的な」）
- [ ] 熱血感・激励感があるか
- [ ] 男性社会人に刺さるか

### STEP4: NG回避チェック
- [ ] 「人の」「やってること」使用していない
- [ ] 「ヤバい」「筋トレ」など危険ワード回避
- [ ] 「瞬間」「証拠」「理由」など不自然な言葉回避
- [ ] 誰のことかわかるか確認

## 💡 作成例（コンテンツ1）

### 元コンテンツの本質
「君には可能性がある、負荷をかけて成長しろ」

### 成功定義
読み手が「成長したい」「行動したい」と思うこと

### タイトル4タイプ
**+タイプ**: 「成長しない人の残念な特徴5選」
**-タイプ**: 「成長できない人」
**×タイプ**: 「成長するための具体的行動」
**÷タイプ**: 「急成長する人」

## 🎯 効果的なパターン集

### 読み手特定型（推奨）
```
「○○できない人の特徴5選」
「○○しない人の習慣5選」
「○○な人がやりがちなこと5選」
```

### KING特有の攻め表現
```
「○○な人の残念な特徴5選」
「○○できない人のダメな習慣5選」
「○○しない人の致命的な原因5選」
```

### 読み手欲求直接型
```
「○○する方法5選」
「○○のコツ5選」
「○○するための習慣5選」
```
