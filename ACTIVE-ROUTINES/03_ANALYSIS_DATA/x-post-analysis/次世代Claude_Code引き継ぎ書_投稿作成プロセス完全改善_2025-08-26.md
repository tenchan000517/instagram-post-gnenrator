## 🏗️ ESSENCE Method システム概要

### 基本設計思想
**「本質優先」** - フレームワークよりコンテンツの核心価値を最優先

### 7ステップシステム
```
E - Extract（抽出）: コンテンツの核心を抽出
S - Specify（特定）: 視聴者の変化を特定
S - Select（選択）: 最適なアプローチを選択
E - Express（表現）: キャラクターらしく表現
N - Name（命名）: 魅力的なタイトルを作成
C - Check（確認）: 品質チェック
E - Execute（実行）: 投稿として完成
```

### 4つの投稿アプローチ
1. **体験談型**: キャラ実体験で共感創出
2. **比較対照型**: Before/After等の対比
3. **問題解決型**: 具体的課題の解決策
4. **気づき型**: 新しい視点・価値観の提示
