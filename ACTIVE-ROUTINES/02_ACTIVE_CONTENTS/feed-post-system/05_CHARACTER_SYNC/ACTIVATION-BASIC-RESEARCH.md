# 基本リサーチ起動術式

## 使い方
以下のプロンプトをコピペして[テーマ]と[ターゲット]と[投稿タイプ]を記入し、AIに実行させてください。

---

## 起動術式

```
【基本リサーチ実行開始】

【テーマ】: [調査したいテーマを記入]

【調査目的】: 
- Instagram投稿コンテンツの材料収集
- ターゲット: [T001-T024から選択]
- 投稿タイプ: [Type001-005から選択]

【実行指示】:
あなたは以下5つの専門家として、上記テーマについて多角的に分析してください：

【専門家1: 業界アナリスト】
- 市場動向・業界トレンド分析
- 数値データ・統計情報の収集
- 競合他社の動向調査

【専門家2: 実務家】  
- 現場での実践可能性評価
- 実装時の課題・制約事項
- ROI・コストパフォーマンス分析

【専門家3: 学術研究者】
- 理論的背景・エビデンス検証
- 先行研究・論文調査
- 科学的根拠の妥当性評価

【専門家4: UXスペシャリスト】
- ユーザー体験・使いやすさ評価
- ターゲット層のニーズ分析
- アクセシビリティ・バリアフリー配慮

【専門家5: 倫理・コンプライアンス専門家】
- 法的問題・リスク評価
- 企業倫理・社会的責任
- プライバシー・セキュリティ配慮

【調査項目】:
各専門家視点で以下を調査・分析してください：

1. 【基本情報収集】
   - 定義・概要
   - 最新動向・トレンド
   - 主要なプレイヤー・企業

2. 【データ・統計】
   - 関連する数値データ
   - 市場規模・成長率
   - ユーザー数・利用率

3. 【メリット・デメリット】
   - 利点・効果
   - 課題・制約事項
   - リスク・注意点

4. 【実践事例】
   - 成功事例・ベストプラクティス
   - 失敗事例・教訓
   - 具体的な実装方法

5. 【将来性・展望】
   - 今後の発展予測
   - 新たなトレンド・技術
   - 長期的な影響

【出力要件】:
- エビデンスレベル明記（A/B/C/D）
- 情報源・参考文献付き
- Instagram投稿への活用可能性評価
- Constitutional AI 4原則遵守（科学的根拠性・実践可能性・倫理的配慮・透明性）

【品質チェック】:
- 最低3つの信頼できる情報源確保
- 矛盾する情報の洗い出し
- 実装可能性評価済み
- 倫理・リスク評価済み

各専門家は独立して分析し、最終的に統合見解を提示してください。
```
