### トレンド評価4軸システム

#### 1. バズ度評価（Virality）
```
【評価基準】
- エンゲージメント率（いいね・シェア・コメント数）
- メディア露出頻度（記事・ニュース掲載数）
- 検索ボリューム推移（Google トレンド）
- ソーシャルメンション数

【スコア算出】
高（80-100点）: 月間1000万以上のインプレッション
中（50-79点）: 月間100万-1000万のインプレッション  
低（0-49点）: 月間100万未満のインプレッション
```

#### 2. 実践度評価（Practicality）
```
【評価基準】
- 一般ユーザーの実践可能性
- 必要リソース・コスト
- 学習・習得の難易度
- 継続実行の容易さ

【実践難易度】
Easy: 特別なスキル・投資不要、即日実践可能
Medium: 基本スキル・少額投資で1週間以内実践可能
Hard: 専門スキル・高額投資で1ヶ月以上習得期間必要
```

#### 3. 持続性評価（Sustainability）
```
【評価基準】
- 一時的ブームか継続的トレンドか
- 類似トレンドの過去実績分析
- 業界専門家の見解・予測
- 基盤技術・制度の安定性

【持続期間予測】
短期（1-3ヶ月）: イベント依存、季節限定
中期（3-12ヶ月）: 技術普及、制度変更
長期（1年以上）: 社会構造変化、価値観変化
```

#### 4. 発展性評価（Scalability）
```
【評価基準】
- 関連分野への波及可能性
- 新たなビジネス機会創出
- イノベーション・改善余地
- グローバル展開可能性

【発展ポテンシャル】
高: 複数業界に影響、新市場創出可能
中: 特定業界内で多様な応用可能
低: 限定的な用途、改善余地少
```

## バズ度・実践度評価システム

### 統合評価マトリクス
```
           実践度
           Easy  Medium  Hard
バズ度 高    A+     A      B+    
      中    A      B+     B
      低    B      C+     C
```

### 評価ランク別活用方針

#### Aランク（優先実装）
```
【特徴】
- 高いバズ度 × 高い実践度
- Instagram投稿として最高の価値

【活用方法】
- Type001（MISAKI）: 感情共感アプローチで共有
- Type002（KING）: 実践ガイドとしてステップ化
- Type004（TEN）: 効率化ツールとして紹介
```

#### Bランク（条件付き実装）
```
【特徴】  
- 中程度のバズ度または実践度
- ターゲット・タイミングを選んで活用

【活用方法】
- 特定ターゲットに限定した投稿
- トレンドの背景・解説コンテンツ
- 将来性にフォーカスした紹介
```

#### Cランク（要検討）
```
【特徴】
- 低いバズ度・実践度
- 現時点では投稿価値限定的

【対応方針】
- モニタリング継続
- 発展性が高い場合は継続観察  
- 他のトレンドとの組み合わせ検討
```

## トレンド分析テンプレート

### 基本分析プロンプト
```
【分析対象トレンド】: [トレンド名を記入]

【分析要請】:
以下の4軸でトレンドを分析してください：

1. 【バズ度評価（0-100点）】
   - 現在のエンゲージメント状況
   - メディア露出度
   - 検索ボリューム推移
   - 根拠となる数値データ

2. 【実践度評価（Easy/Medium/Hard）】
   - 一般ユーザーの実行可能性
   - 必要なリソース・コスト
   - 習得期間・難易度
   - 実践時の障壁・課題

3. 【持続性評価（短期/中期/長期）】
   - 一時的ブームか継続トレンドか
   - 類似トレンドの過去実績
   - 専門家の見解・予測
   - 持続を支える要因分析

4. 【発展性評価（高/中/低）】
   - 関連分野への波及可能性
   - 新たなビジネス機会
   - イノベーション創出余地
   - グローバル展開ポテンシャル

【Instagram投稿活用提案】:
- 最適なType（001-005）の選択理由
- ターゲット層（T001-T024）の推奨
- コンテンツ構成案（8ページ）
- 期待されるエンゲージメント予測
```
