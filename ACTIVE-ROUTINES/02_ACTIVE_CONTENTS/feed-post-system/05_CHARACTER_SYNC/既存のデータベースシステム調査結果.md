# 汎用データベース拡張マスターシステム（完全統合版）

**作成日**: 2025-08-28  
**目的**: 習慣・企業・全ジャンルのデータベース拡張を統一フローで実現  
**基盤**: 習慣系4段階 + 企業系5段階の統合  

---

## 🎯 システム概要

### 統一データベース拡張フロー

```
【汎用4段階フロー】

①究極リサーチ段階
  → 新分野の専門家集団生成・外部調査
  → エビデンスレベルA/B限定の高品質データ収集

②データベース統合段階  
  → 既存データベースへの構造化追加
  → JSON構造準拠・品質保証

③ランキング・コンテンツ生成段階
  → データベース活用のランキング・分析コンテンツ作成
  → キャラクター別最適化

④評価・分析・改善段階
  → 品質チェック・継続的改善
  → 次回拡張の計画策定
```
