# 📊 Instagram フィード投稿システム - 新規作成フロー完全版

## 🎯 システム概要
Instagram投稿の新規作成に特化した6ステップフロー。リサーチから完成まで一気通貫で高品質投稿を生成。

## 🗂️ ディレクトリ構造と実行順序

```
feed-post-system/
├── 00_SYSTEM_MASTER/           ⭐ システム定義・設計書
├── 01_RESEARCH_PROMPTS/        ① リサーチプロンプトと起動術式
├── 02_TYPE_ACTIVATION/         ② 投稿タイプ別起動術式
├── 03_CONTENT_CREATION/        ③ コンテンツ作成マスターガイドライン
├── 04_TEMPLATE_PLACEMENT/      ④ テンプレート配置マスターガイドライン
├── 05_CHARACTER_SYNC/          ⑤ キャラクター別シンクロドキュメント
├── 06_CAPTION_HASHTAG/         ⑥ キャプション・ハッシュタグ作成マニュアル
├── 07_REFERENCE_MATERIALS/     📚 重要リファレンス材料
└── 99_ARCHIVE/                 📦 アーカイブ（旧システム）
```

## 🚀 新規作成フロー（6ステップ）

### **Step 1: リサーチ・ネタ収集**
📁 `01_RESEARCH_PROMPTS/`
- トレンド調査

- **Type003**: KIKUYO（ランキング情報）
- **Type004**: TEN（効率化テクニック）
- **Type005**: IIDA（就活サポート）

### **Step 3: コンテンツ作成**
📁 `03_CONTENT_CREATION/`
- Type別品質基準適用
- コンテンツ構造設計
- 価値提供最大化

### **Step 4: テンプレート配置**
📁 `04_TEMPLATE_PLACEMENT/`
- 最適テンプレート選択
- ページ構成最適化
- 情報階層設計

### **Step 5: キャラクターシンクロ**
📁 `05_CHARACTER_SYNC/`
- キャラクター個性統一
- トーン&マナー調整
- 読者との関係性構築

### **Step 6: キャプション・ハッシュタグ**
📁 `06_CAPTION_HASHTAG/`
- エンゲージメント最大化文言
- ターゲット別ハッシュタグ戦略
- 発見性・保存率向上
