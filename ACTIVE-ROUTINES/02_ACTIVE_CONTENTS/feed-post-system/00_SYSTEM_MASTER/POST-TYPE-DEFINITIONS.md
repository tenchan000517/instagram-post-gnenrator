# 📋 Instagram投稿タイプ定義書 v2.0

## 🎯 投稿タイプ一覧（5タイプ）

### **Type001: 感情共感型**
- **メインキャラクター**: MISAKI
- **対象**: キャリアやライフスタイルに悩みを持った女性
- **特徴**: 感情共感→安心→希望のフロー
- **テンプレート**: section-blocks構造（2セクション）

### **Type002: 実践習得型**  
- **メインキャラクター**: KING
- **対象**: 社会人男性（20代～40代）
- **特徴**: 転職も視野に入れつつ、今の仕事に悩みを感じている人向け
- **テンプレート**: 8ステップ思考プロセス

### **Type003: ランキング情報型**
- **メインキャラクター**: KIKUYO
- **対象**: 全般（情報収集志向）
- **特徴**: 
  - 業界分析、習慣ランキング、人気転職ランキング等
  - データベース使用のランキング・Tier表
  - リール投稿のデータベースロジックと同じ
- **重点**: データ抽出方法とテンプレート配置

### **Type004: 効率化テクニック型**
- **メインキャラクター**: TEN
- **対象**: 女性・男性・学生問わず
- **特徴**: 
  - AI、ツール、ガジェット、アイテム活用
  - 生産性向上・生活の豊かさ追求
- **テンプレート**: 実証型アプローチ

### **Type005: 就活サポート型【新設】**
- **メインキャラクター**: IIDA
- **対象**: 学生（就活生）
- **特徴**: 
  - 就活に関する悩み解決
  - ティップス・ノウハウ・情報発信
- **テンプレート**: 未定（要設計）
