# トレンド分析起動術式

## 使い方
以下のプロンプトをコピペして[トレンド名]を記入し、AIに実行させてください。

---

## 起動術式

```
【トレンド分析実行開始】

【分析対象トレンド】: [トレンド名を記入]

【分析要請】:
以下の4軸でトレンドを分析してください：

1. 【バズ度評価（0-100点）】
   - 現在のエンゲージメント状況
   - メディア露出度
   - 検索ボリューム推移（Google トレンド）
   - ソーシャルメンション数
   - 根拠となる数値データ

   【スコア基準】
   - 高（80-100点）: 月間1000万以上のインプレッション
   - 中（50-79点）: 月間100万-1000万のインプレッション  
   - 低（0-49点）: 月間100万未満のインプレッション

2. 【実践度評価（Easy/Medium/Hard）】
   - 一般ユーザーの実行可能性
   - 必要なリソース・コスト
   - 習得期間・難易度
   - 実践時の障壁・課題

   【難易度基準】
   - Easy: 特別なスキル・投資不要、即日実践可能
   - Medium: 基本スキル・少額投資で1週間以内実践可能
   - Hard: 専門スキル・高額投資で1ヶ月以上習得期間必要

3. 【持続性評価（短期/中期/長期）】
   - 一時的ブームか継続トレンドか
   - 類似トレンドの過去実績
   - 専門家の見解・予測
   - 持続を支える要因分析

   【持続期間予測】
   - 短期（1-3ヶ月）: イベント依存、季節限定
   - 中期（3-12ヶ月）: 技術普及、制度変更
   - 長期（1年以上）: 社会構造変化、価値観変化

4. 【発展性評価（高/中/低）】
   - 関連分野への波及可能性
   - 新たなビジネス機会
   - イノベーション創出余地
   - グローバル展開ポテンシャル

   【発展ポテンシャル】
   - 高: 複数業界に影響、新市場創出可能
   - 中: 特定業界内で多様な応用可能
   - 低: 限定的な用途、改善余地少

【統合評価マトリクス判定】:
           実践度
           Easy  Medium  Hard
バズ度 高    A+     A      B+    
      中    A      B+     B
      低    B      C+     C

【Instagram投稿活用提案】:
- 最適なType（001-005）の選択理由
- ターゲット層（T001-T024）の推奨
- コンテンツ構成案（8ページ）
- 期待されるエンゲージメント予測

【情報収集ソース】:
- LinkedIn トレンディング投稿
- Twitter/X トレンディングトピック
- Google トレンド検索ボリューム
- YouTube 急上昇ランキング
- 業界専門メディア（日経、東洋経済、Diamond）
- 海外トレンド（Harvard Business Review、Forbes）

【出力形式】:
- 各軸の評価点数・レベル明記
- 統合評価ランク（A+/A/B+/B/C+/C）
- Instagram活用戦略
- エビデンス・参考文献
```
