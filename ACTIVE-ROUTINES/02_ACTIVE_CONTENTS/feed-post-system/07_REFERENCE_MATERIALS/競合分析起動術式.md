# 競合分析起動術式

## 使い方
以下のプロンプトをコピペして[企業名・業界]を記入し、AIに実行させてください。

---

## 起動術式

```
【競合分析実行開始】

【分析対象企業】: [企業名・業界を記入]

【分析要請】:
以下の観点で競合分析を実施してください：

1. 【基本情報収集】
   - 企業概要（規模・事業内容・沿革）
   - 財務状況（売上・利益・成長率）
   - 市場ポジション（シェア・競争優位性）
   - 組織構造（従業員数・拠点・子会社）

2. 【戦略・事業分析】
   - 事業戦略・成長戦略
   - 主要サービス・製品ラインナップ
   - 技術・イノベーション投資
   - M&A・提携戦略

3. 【組織・人材分析】
   - 企業文化・価値観
   - 人材育成・キャリア制度
   - 働き方改革・福利厚生
   - ダイバーシティ・インクルージョン

4. 【マーケティング・ブランド分析】
   - ブランド戦略・ポジショニング
   - マーケティング手法・チャネル
   - 顧客セグメント・ターゲティング
   - PR・広報戦略

5. 【ベストプラクティス抽出】
   - 学習すべき成功要因
   - 実装可能性評価（Easy/Medium/Hard）
   - 文化適合性評価（高/中/低）
   - ROI分析・投資対効果

【SWOT分析実行】:

【Strengths（強み）】
- 独自技術・ノウハウ
- ブランド力・認知度
- 財務体質・資金力
- 人材・組織力

【Weaknesses（弱み）】
- 技術・サービスの限界
- コスト構造の問題
- 組織・文化の課題
- 市場ポジションの弱さ

【Opportunities（機会）】  
- 市場拡大・新規参入機会
- 技術革新・トレンド活用
- 規制緩和・制度変更
- パートナーシップ・M&A

【Threats（脅威）】
- 競合参入・価格競争
- 技術陳腐化・破壊的革新
- 規制強化・コンプライアンス
- 経済変動・市場縮小

【実装可能性評価】:

【レベル1: 即座実装可能（Easy）】
- 追加投資・リソース不要
- 既存システム・プロセスで対応可能
- 1週間以内に実装・効果発現

【レベル2: 条件付き実装可能（Medium）】
- 限定的な投資・リソース必要
- 部分的なシステム・プロセス変更
- 1ヶ月以内に実装・3ヶ月で効果発現

【レベル3: 長期計画必要（Hard）】
- 大規模投資・リソース必要
- 抜本的なシステム・プロセス変革
- 6ヶ月以上の実装期間・1年で効果発現

【文化適合性評価（日本企業特化）】:

【組織文化適合性（1-5点）】
- 既存の権威構造との整合性
- 合意形成プロセスへの配慮
- 段階的実装の可能性
- 現場の抵抗最小化

【コミュニケーション文化適合性（1-5点）】
- 暗黙知の明文化
- 対立を避けた実装方法
- 関係性構築への配慮
- デジタル化への配慮

【変革態度適合性（1-5点）】
- リスクの適切な管理
- 類似事例・前例の提示
- 段階的実装計画
- 関係者の合意形成考慮

【総合評価】
- 高適合（12-15点）: 積極実装推奨
- 中適合（8-11点）: 条件調整後実装
- 低適合（3-7点）: 大幅修正または見送り

【Instagram投稿への活用】:
- 差別化ポイントの明確化
- 競合優位性のアピール方法
- ベンチマークとしての活用
- トレンド先取りのヒント

【出力要件】:
- 全項目の詳細分析
- SWOT分析結果
- 実装可能性・文化適合性スコア
- Instagram活用戦略
- エビデンス・参考文献
```
