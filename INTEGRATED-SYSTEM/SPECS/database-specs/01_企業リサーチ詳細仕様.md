# 企業リサーチ詳細仕様書

## 実行概要
**起動術式**: `INTEGRATED-SYSTEM/LAUNCH/database/01_企業リサーチ実行.md`
**対象業界**: 23業界（IT、金融、商社、メーカーなど）
**生成ランキング数**: 233+パターン

## 詳細仕様

### データ収集範囲
- **対象企業数**: 業界別上位100社
- **データ項目**: 年収、初任給、年間休日、残業時間、勤続年数、有給取得率、離職率
- **情報源**: 企業公式、就職四季報、転職サイト、口コミサイト

### ランキング生成パターン

#### femaleCareer系（女性キャリア特化）
- FC001-030: ワークライフバランス企業TOP10-20
- 年間休日130日以上、残業20時間以内
- 有給取得率80%以上、男性育休取得率50%以上
- 平均勤続年数15年以上

#### jobSeekers系（就活生向け）
- JS001-030: 初任給ランキングTOP10-20
- 3年離職率が低い企業TOP10
- 総合スコア企業TOP10
- 業界別初任給・年間休日ランキング

#### maleProfessional系（男性社会人向け）
- MP001-030: 年収ランキングTOP10-20
- 年収1500万円以上企業TOP10
- 転職価値総合ランキングTOP15
- 高年収×働きやすさバランスTOP10

### データ品質基準
- **信頼性**: 企業公式データ優先
- **更新頻度**: 年次更新必須
- **検証方法**: 複数情報源でのクロスチェック
- **エラー許容範囲**: 数値データ±3%以内

### 実行手順
1. 業界別企業リスト作成（上位100社）
2. データ項目別情報収集
3. ランキング生成（233パターン）
4. JSON形式でデータベース保存
5. 品質チェック・検証

### 関連ファイル
- **データ保存先**: `app/data/companyDatabase/`
- **業界別データ**: `industries/[業界名]_companies.json`
- **ランキングデータ**: `rankingsV2/[カテゴリ]/`
- **生成スクリプト**: `generateAllRankingsV2.js`

### トラブルシューティング
- **データ不足**: 代替情報源への切り替え
- **ランキング重複**: 一意性チェック実装
- **処理エラー**: ログ出力・段階的デバッグ

## 成果物
- 233+個のランキングJSONファイル
- 23業界別企業データベース
