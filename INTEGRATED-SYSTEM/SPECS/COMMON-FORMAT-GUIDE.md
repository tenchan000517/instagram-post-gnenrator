# Instagram投稿共通フォーマットガイド

**対象**: 全キャラクター（MISAKI・KING・KIKUYO・TEN・IIDA）  
**適用範囲**: リール動画・フィード投稿の基本フォーマット  
**目的**: キャラクター固有要素を除く共通「型」の統一

---

## 🎬 リール動画共通フォーマット

### **基本構造（60秒固定）**
```
【フック部分】0-10秒
- 注意喚起・問題提起
- 保存促進CTA
- インパクトある導入

【メイン部分】10-50秒  
- 核心コンテンツ
- キャラクター固有の構成（2-10ページ）
- 具体的情報提示

【クロージング部分】50-60秒
- 行動促進
- コメント促進
- 次回予告（任意）
```

### **文字数制限ルール**
- **各行**: 15-20文字以内
- **1画面**: 最大3行まで
- **読み上げ時間**: 1行あたり2-3秒

### **必須CTA要素**
```
✅ フック部分: 「後で見返せるよう今すぐ保存して」
✅ クロージング部分: 「コメントで教えて」「質問があれば」
```

---

## 📱 フィード投稿共通フォーマット

### **基本構造（8枚カルーセル固定）**
```
1枚目: タイトル・キャッチ
2-7枚目: メインコンテンツ（キャラクター固有構成）
8枚目: まとめ・CTA
```

### **JSON構造（共通部分）**
```json
{
  "source": "[固有ID]",
  "problemCategory": "[カテゴリ名]",
  "knowledgeId": "[ID]",
  "postType": "[001-005]",
  "pageCount": 8,
  "pageStructurePattern": "[テンプレート名]",
  "targetIds": ["[ターゲットID]"],
  "detailedContent": {
    // キャラクター固有のページ構成
  }
}
```

---

## 💬 キャプション共通フォーマット

### **基本構造**
```
【導入部】（40-60文字）
[問題提起・共感・状況設定]

【メイン部】（200-300文字）
[核心メッセージ・価値提供]

【CTA部】（30-50文字）
[行動促進・コメント促進]

【ハッシュタグ】
#[メインタグ] #[関連タグ1] #[関連タグ2] #[キャラクタータグ]
```

### **ハッシュタグ規則**
- **メインタグ**: コンテンツテーマ
- **関連タグ**: 2-3個まで
- **キャラクタータグ**: 各キャラクター固有
- **総数**: 5-8個以内

---

## ⛔ 禁止表現・品質基準

### **絶対禁止語句**
```
❌ システム、戦略的、科学的、技術、最適化
❌ 効率的、効率化、改善、向上
❌ 方法論、テクニック、ノウハウ
❌ メソッド、フレームワーク、プロセス
```

### **機械的表現禁止**
```
❌ 「1.」「2.」「3.」の箇条書き
❌ 「ステップ1:」「ステップ2:」
❌ 「第一に」「第二に」「最後に」
❌ 「以下の通りです」「下記をご覧ください」
```

### **AI生成感排除ルール**
- ビジネス文書調の排除
- 不自然な敬語の排除  
- 機械的な接続詞の削減
- 自然な話し言葉への変換

---

## 📝 文字数・文章品質基準

### **リール動画用**
- **1行**: 15-20文字以内
- **1画面**: 最大3行
- **総文字数**: 400-600文字

### **フィード投稿用**
- **タイトル**: 10-15文字
- **説明文**: 1セクション50-100文字
- **総文字数**: 800-1200文字

### **キャプション用**
- **総文字数**: 300-500文字
- **段落分け**: 3-4段落
- **読みやすさ**: 改行・絵文字活用

【フィード投稿】
