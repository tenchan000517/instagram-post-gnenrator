## 📋 リール動画台本構造

### 1. フック部分（0-10秒）

#### 構成要素
```
【0-3秒】タイトル提示
【3-6秒】煽り・危機感演出  
【6-10秒】価値提供＋保存促進CTA
```

#### 共通テンプレート
```
今すぐ[導入/チェック]すべき[ジャンル]TOP5

これ知らないと損して[ます/るのです]

特に1位は[具体的効果]
後で見返せるよう今すぐ保存して[ください]
```

---

### 2. ランキング部分（10-50秒）

#### 時間配分
```
各ランキング: 8秒×5項目 = 40秒
5位: 10-18秒
4位: 18-26秒  
3位: 26-34秒
2位: 34-42秒
1位: 42-50秒
```

---

## 🤖 TEN版ナレーション構造（5行テンプレート）

### 構成
```
第X位：[ツール名]
[AIを一言で説明]
[機能説明1行目]
[機能説明2行目]
[効果説明1行目]
[効果説明2行目]
```

### 各行の作成ルール

#### 1行目：ランキング表示
- 形式: 「第X位：[ツール名]」
- ツール名: カタカナ表記（パープレキシティ、ジェミニ等）

#### 2行目：一言説明
- 文字数: 10-15文字
- 例: 「最新の検索特化型AI」

#### 3-4行目：機能説明
- 各行文字数: 15-20文字
- 3行目: 主要機能・処理能力
- 4行目: 特徴的機能・差別化要素

#### 5-6行目：効果説明
- 各行文字数: 15-20文字
- 5行目: 直接的効果・改善点
- 6行目: 具体的な時短・効率化効果

## ⚠️ 品質基準・注意事項

### 必須チェック項目
- [ ] 60秒以内の時間配分
- [ ] 正式名称の使用（投稿用）
- [ ] カタカナ表記（ナレーション）
- [ ] 改行位置の正確性
- [ ] キャラクター口調の一貫性

### TEN版の注意点
- 5行構造を維持
- 効率化・生産性の観点を強調
- 「〜なり」「〜であるぞ」等の古風な表現

### KIKUYO版の注意点
- 2-3行の簡潔構造（5位・4位・2位は2行、3位・1位は3行）
- 「のです」は原則使用しない
- 自然で聞きやすい日本語を優先
- 冒頭の「これ知らないと損するのです」を削除（簡潔化）

### 避けるべき表現
- ❌ 曖昧な効果表現
- ❌ 誇張表現
- ❌ 複雑な専門用語
- ❌ 長すぎる文章（各行20文字超過）
- ❌ 「のです」の過剰使用（KIKUYO版）
