```
【KIKUYO リール動画作成実行】

ランキングテーマ: [例：IT企業TOP5、女性が活躍している企業TOP5...]

```

```
INTEGRATED-SYSTEM/RESOURCES/creation-manuals/REEL-VIDEO-MASTER-PROMPT-V2.md
```

### 2. **データソース（企業ランキングJSON）**
```
app/data/knowledgeBase/knowledge/type003/KY0XX.json
```

---

## 🛠️ リール動画作成手順（4ステップ）

### **STEP1: データソース確認**
- [ ] KY001-KY999のKIKUYO専用JSONから対象ファイルを選択
- [ ] TOP10企業ランキングからTOP5を抽出
- [ ] 各企業の基本情報（企業名・特徴・強み）を確認

### **STEP2: KIKUYO版構造作成**
#### **フック部分（0-10秒）**
```
今すぐチェックすべき[ジャンル]TOP5

特に1位は[具体的効果]
後で見返せるよう今すぐ保存してください
```

#### **ランキング部分（10-50秒）**
```
5位・4位・2位：2行説明
3位・1位：3行説明（重要項目として差別化）
```

#### **クロージング部分（50-60秒）**
```
今すぐチェックしてみて
```

### **STEP3: KIKUYO版ナレーション作成**
#### **2行構成（5位・4位・2位）**
```
第X位：[企業名]
[企業の特徴・主要事業]
[強み・成長性・差別化要素]
```

#### **3行構成（3位・1位）**
```
第X位：[企業名]
[主要製品・サービス/事業範囲]
[企業の特徴・位置づけ/先端技術]
[強み・成長性/業界での地位・最高峰理由]
```

#### **重要ルール**
- **「のです」は原則使用しない**（自然な日本語優先）
- **各行15-20文字以内**
- **投稿用は正式名称（英語表記）使用**

### **STEP4: 最終出力・保存**
- [ ] 60秒構成の確認
- [ ] KIKUYO簡潔版フォーマット準拠確認
- [ ] 保存先: `INTEGRATED-SYSTEM/OUTPUT/reel-posts/KIKUYO/kikuyo_reel_###.md`
