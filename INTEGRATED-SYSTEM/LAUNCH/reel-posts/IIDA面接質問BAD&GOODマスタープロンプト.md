## 使用方法

### 1. 投稿作成フロー
1. `/INTEGRATED-SYSTEM/LAUNCH/reel-posts/IIDA投稿リスト_パターン1_3企画.md` から投稿を選択
2. データベースから該当する3問のデータを取得
3. 投稿用フォーマットで13スライドのコンテンツ作成
4. キャプション用フォーマットで完全固定フォーマットのテキスト作成  
5. ナレーション用フォーマットで読み上げ原稿作成

### 2. データ取得方法
```javascript
// データベースから質問データ取得
const questionData = {
  post_title: "面接で絶対に聞かれる質問【第1弾】",
  question1: database.questions.find(q => q.id === "Q001").question,
  bad_answer1: database.questions.find(q => q.id === "Q001").bad_answer,
  good_answer1: database.questions.find(q => q.id === "Q001").good_answer,
  // Q002, Q003も同様
};
```

### 3. 品質チェック項目
- [ ] データベース内容をそのまま使用しているか
- [ ] 投稿リストのタイトルと一致しているか
- [ ] キャプション固定フォーマット遵守
- [ ] ナレーションに読み上げない要素が含まれていないか

これらのフォーマットにより、データベース準拠の高品質な投稿を効率的に生成できます。
