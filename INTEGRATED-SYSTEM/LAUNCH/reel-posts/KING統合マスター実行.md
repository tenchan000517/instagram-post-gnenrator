## 🛠️ 統合作成手順（6ステップ）

### **STEP1: 基盤情報収集・最適化準備**
- [ ] 指定されたコンテンツ番号の元コンテンツを読み取り
- [ ] KINGらしいタイトルパターンを選択
- [ ] 元コンテンツの成果・実績メッセージを理解
- [ ] **最適化モード起動**: KING品質基準を適用

### **STEP2: 10ページ構成設計（KING最適化）**
**基本構成**:
```
1ページ目: インパクトタイトル
2ページ目: 元コンテンツ（KINGらしい強い言葉・数字強調）
3ページ目: 成果比較表（数値・実績形式）
4ページ目: 改めてタイトル（行動喚起）
5-9ページ目: 成功法則（具体的戦略・実践手法）
10ページ目: CTA（固定・作成不要）
```

### **STEP3: 成果比較表作成**
**形式**:
```
成果が出せない人 vs 圧倒的成果を出す人

[失敗パターンタイトル]
項目1（停滞する行動）
項目2（妥協する選択）
...項目8

[成功パターンタイトル] 
項目1（成長する行動）
項目2（挑戦する選択）
...項目8
```

**KING最適化ルール**:
- [ ] 数字・成果を具体的に表現
- [ ] 行動の結果を明確化
- [ ] 12-15文字で力強い表現
- [ ] 競争・成長を意識した内容

### **STEP4: 成功法則ページ作成（5-9ページ）**

#### **タイトル最適化4原則（KING版）**
```
1. 成果直結表現：結果・効果を前面に
2. 動詞強化：「実行する」「達成する」等
3. 数字活用：「3倍」「TOP10%」等
4. 10-15文字：インパクト重視
```

#### **KINGページ構成テンプレート**
```
### Xページ目: 成功法則X
[インパクトタイトル]（10-15文字・成果強調）

[説明文4行・戦略的内容]
- 1行目: 課題認識（「〜という壁がありますが」）
- 2行目: 戦略提示（「〜を実行すると」）
- 3行目: 具体的成果（「〜を達成でき」）
- 4行目: 更なる効果（「〜になります」）
```

#### **自動適用ルール**
- [ ] **成果数値化**: 「良くなった」→「生産性30%向上」
- [ ] **戦略的表現**: 「やってみる」→「戦略的に実行する」
- [ ] **競争意識**: 「できる」→「他者を圧倒する」
- [ ] **KINGトーン**: 力強い・自信に満ちた・成果志向

### **STEP5: KINGナレーション作成**

#### **構造（成果重視版）**
```
段落1: 導入（50-70文字）
[ビジネス課題]で悩んでいませんか？[成果の差]が出る理由、それは[戦略の違い]なんです。

段落2: 成果説明（30-40文字）  
実際、[具体的数値成果]を達成できます。

段落3: 対比説明（70-90文字）
成果が出ない人は[失敗行動]。一方、圧倒的成果を出す人は[成功行動]を徹底しています。

段落4: タイトル提示（力強く）
今日お伝えするのは「[成功法則タイトル]」。

段落5: 5つの戦略（各35-45文字・実践的）
第1に[戦略1]。最初は[困難]ですが、[成果数値]。
第2に[戦略2]。[場面]では、[具体的行動]で[結果]。
第3に[戦略3]。[期間]で[測定可能な成果]を実現。
第4に[戦略4]。[投資]することで[リターン]を獲得。
最後に[戦略5]。[継続]により[長期的成果]を確立。

段落6: CTA（40-50文字）
成果を出したい方は、今すぐ保存して実践してください。質問があればコメントで！
```

#### **ナレーション最適化ルール**
- [ ] **数値・実績強調**: 具体的成果を前面に
- [ ] **行動喚起**: 「実践する」「挑戦する」等
- [ ] **競争意識喚起**: 「他者との差」を意識
- [ ] **総文字数550文字以内**
- [ ] **力強いテンポ**: 自信に満ちた語り

### **STEP6: キャプション生成**
**重要**: 必ず以下のマニュアルを熟読してから作成
```
INTEGRATED-SYSTEM/RESOURCES/creation-manuals/10_キャプション・ハッシュタグ生成マニュアル_2025-08-23.md
```
- [ ] マニュアルの「KINGタイプ」セクションを確認
- [ ] 正しいフォーマットで生成（熱血激励型）
- [ ] CTAは含めない（10ページ目は固定で作成不要）

### **STEP7: 統合完成・保存**
- [ ] リール動画10ページ構成完成（10ページ目は固定・作成不要）
- [ ] KINGらしい力強いメッセージ確認
- [ ] 男性社会人への訴求力チェック
- [ ] キャプション生成完了（マニュアル準拠）
