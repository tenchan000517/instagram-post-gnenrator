## 🛠️ リール動画作成手順（4ステップ）

### **STEP1: データソース確認**
- [ ] TxxxのTEN専用JSONから対象ファイルを選択
- [ ] TOP10ツールランキングからTOP5を抽出
- [ ] 各ツールの基本情報（ツール名・機能・効果）を確認

### **STEP2: TEN版構造作成**
#### **フック部分（0-10秒）**
```
今すぐ導入すべき[ジャンル]TOP5

これ知らないと損してます

特に1位は[具体的効果]
後で見返せるよう今すぐ保存して
```

#### **ランキング部分（10-50秒）**
```
全項目：6行説明（TEN版フル構成）
各項目8秒×5項目 = 40秒配分
```

#### **クロージング部分（50-60秒）**
```
今日から1つでも導入してみて
```

### **STEP3: TEN版ナレーション作成**
#### **6行構成（全項目共通）**
```
第X位：[ツール名]
[AIを一言で説明]
[機能説明1行目]
[機能説明2行目]
[効果説明1行目]
[効果説明2行目]
```

#### **重要ルール**
- **各行15-20文字以内**
- **投稿用は正式名称（英語表記）使用**
- **ナレーション用はカタカナ表記**

### **STEP4: 最終出力・保存**
- [ ] 60秒構成の確認
- [ ] TEN6行版フォーマット準拠確認
- [ ] 保存先: `INTEGRATED-SYSTEM/OUTPUT/reel-posts/TEN/ten_reel_###.md`
