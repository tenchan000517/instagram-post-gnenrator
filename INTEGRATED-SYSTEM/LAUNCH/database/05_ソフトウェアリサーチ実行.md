## 🔍 必須リサーチ項目（35項目）

### **基本情報（9項目）**
1. **id**: ソフトウェアID（SW001-SW060）
2. **softwareName**: ソフトウェア名
3. **category**: カテゴリ分類
4. **companyName**: 開発会社名
5. **officialUrl**: 公式サイトURL
6. **releaseYear**: サービス開始年
7. **latestUpdate**: 最新大型アップデート日
8. **serviceType**: サービス形態（SaaS/Desktop/Mobile/Hybrid）
9. **userBase**: ユーザー規模・企業導入実績

### **料金・プラン情報（8項目）**
10. **freeTier**: 無料プラン詳細・制限
11. **personalPlan**: 個人プラン価格・機能
12. **teamPlan**: チームプラン価格・機能
13. **enterprisePlan**: 企業プラン価格・機能
14. **billingOptions**: 支払い方式（月額/年額/買い切り）
15. **freeTrialPeriod**: 無料試用期間
16. **studentDiscount**: 学生・教育割引
17. **roiCalculation**: 投資回収期間・コスト効果

### **機能・技術仕様（10項目）**
18. **coreFeatures**: 主要機能（5-7個）
19. **uniqueFeatures**: 独自・差別化機能
20. **platformSupport**: 対応プラットフォーム
21. **mobileApp**: モバイルアプリ品質
22. **offlineCapability**: オフライン機能
23. **integrations**: 主要連携サービス
24. **apiAvailability**: API提供・拡張性
25. **securityFeatures**: セキュリティ機能
26. **dataExport**: データエクスポート・移行
27. **customization**: カスタマイズ可能性

### **使用性・効果情報（8項目）**
28. **setupTime**: 初期設定・導入時間
29. **learningCurve**: 習得難易度（Easy/Medium/Hard）
30. **teamAdoptionTime**: チーム導入・浸透期間
31. **productivityIncrease**: 生産性向上率（定量的）
32. **userSatisfaction**: ユーザー満足度・評価
33. **supportQuality**: サポート品質・レスポンス
34. **japaneseSupport**: 日本語対応レベル
35. **commonUseCases**: 主な活用事例・業務シーン

## TEN 6項目評価基準（ソフトウェア特化版）

### 評価算出式
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### 各項目評価基準

#### **即効性 (20%) - 効果実感の早さ**
- 100点: 導入当日から業務効率向上を実感
- 90点: 1週間以内に明確な効果を実感
- 80点: 2-3週間で効果を実感・ワークフロー改善
- 70点: 1ヶ月以内に効果を実感・習慣化完了
- 60点: 2-3ヶ月で効果を実感・チーム浸透

#### **簡単さ (20%) - 導入・使用の容易さ**
- 100点: 誰でも即座に利用開始・直感的操作
- 90点: 数分で基本設定完了・学習コスト最小
- 80点: 1時間程度で基本機能習得・簡単操作
- 70点: 半日程度で習得・一般的な学習コスト
- 60点: 数日の学習・設定が必要・やや複雑

#### **人気度 (15%) - 市場シェア・認知度**
- 100点: 業界標準・圧倒的シェア・誰もが知る
- 90点: 高いシェア・多くの企業で採用
- 80点: 業界で評価・専門分野で高い認知
- 70点: 一定のシェア・成長中・評価良好
- 60点: ニッチ分野・特定用途で評価

#### **コスパ (15%) - 費用対効果**
- 100点: 無料で高機能・圧倒的なコスパ
- 90点: 低価格で十分な機能・優秀なコスパ
- 80点: 適正価格で相応の価値・標準的コスパ
- 70点: やや高価格だが納得できる価値
- 60点: 高価格だが独自価値・限定的コスパ

#### **機能専門性 (20%) - 専門性・差別化要素**
- 100点: 他にない独自機能・圧倒的な専門性
- 90点: 高度な専門機能・明確な競合優位
- 80点: 優れた専門機能・特定分野で強み
- 70点: 一般的専門機能・標準以上の性能
- 60点: 基本的機能・特別な差別化なし

#### **生産性UP度 (10%) - 実際の業務効率向上効果**
- 100点: 劇的な業務効率化・時間短縮・品質向上
- 90点: 明確な効率向上・作業時間大幅短縮
- 80点: 一定の効率向上・ワークフロー改善
- 70点: 軽度の効率向上・作業品質改善
- 60点: 限定的な効率向上・主観的満足

## 品質基準
- データ精度：公式機能・実際の料金データ優先
- 実用性効果：実際の業務効率向上・チーム導入成功事例重視  
- 日本市場：日本でのサポート品質・利用状況を評価基準に
- 最新性：2025年最新機能・料金動向反映
- 継続性：長期利用・チーム展開の実現性を検証
```
