## **⚠️ 絶対遵守事項（違反時は作業失敗）**

### **データ入力の絶対ルール**
1. **数値は全て半角数字**（全角禁止）
2. **金額の単位は全て円**（「万円」等の表記禁止）
3. **パーセントは数値のみ**（%記号を含めない）
4. **期間は全て月単位**（年は12倍して月に変換）
5. **nullと0は明確に区別**（データなし=null、ゼロ=0）
6. **文字列は全てダブルクォート**で囲む
7. **公式サイト情報のみ使用**（推測データ禁止）

### **5軸評価システム（各1-5の整数）**
1. **取得難易度（difficultyLevel）**: 
   - 1: 講習のみ、2: 講習＋簡単テスト、3: 独学可能、4: 専門教育必要、5: 超難関
2. **専門性（specialization）**: 
   - 1: 汎用基礎、2: 特定入門、3: 実務知識、4: 専門知識、5: 高度専門
3. **キャリア価値（careerValue）**: 
   - 1: ほぼ考慮されない、2: 少し有利、3: 特定業界重視、4: 必須レベル、5: 複数業界必須
4. **収入インパクト（incomeImpact）**: 
   - 1: 影響なし、2: 専門業務可能、3: 資格手当1-3万、4: 高給600万以上、5: 独立1000万可能
5. **取得しやすさ（acquisitionEase）**: ⭐**重要：高いほど良い**
   - 1: 極難取得（100万円超・2年超）、2: 取得困難（100万以下・2年以内）、3: 中程度（20万以下・6ヶ月以内）、4: 取得しやすい（5万以下・3ヶ月以内）、5: 極易取得（1万以下・1ヶ月以内）

### **総合スコア計算**
```
totalScore = difficultyLevel + specialization + careerValue + incomeImpact + acquisitionEase
```
**最小値：5、最大値：25**
