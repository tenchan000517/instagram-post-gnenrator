## 🔍 必須リサーチ項目（28項目）

### **基本情報（6項目）**
1. **id**: 習慣ID（HB001-HB060）
2. **habitName**: 習慣名
3. **category**: カテゴリ分類
4. **difficulty**: 難易度レベル（1-5段階）
5. **timeRequired**: 必要時間（分/日）
6. **frequencyType**: 実施頻度（毎日/週3回等）

### **科学的根拠・効果（8項目）**
7. **scientificBasis**: 科学的根拠・研究データ
8. **primaryBenefits**: 主要効果（3-5個）
9. **secondaryBenefits**: 副次効果
10. **measurableOutcomes**: 測定可能な成果指標
11. **timeToSeeResults**: 効果実感までの期間
12. **longTermImpact**: 長期的な影響・効果
13. **researchReferences**: 参考研究・論文
14. **expertRecommendations**: 専門家の推奨度

### **実践・継続性（8項目）**
15. **startingSteps**: 開始手順・初期ステップ
16. **continuationTips**: 継続のコツ・秘訣
17. **commonChallenges**: よくある挫折要因
18. **solutionStrategies**: 問題解決戦略
19. **progressTracking**: 進捗確認方法
20. **motivationMaintenance**: モチベーション維持法
21. **habitStacking**: 他習慣との組み合わせ
22. **environmentSetup**: 環境設定・準備

### **実用性・適用性（6項目）**
23. **targetPersonas**: 適用対象者（学生/社会人/主婦等）
24. **seasonality**: 季節・時期による影響
25. **culturalAdaptation**: 日本文化への適応性
26. **costRequirement**: 費用・投資必要性
27. **equipmentNeeded**: 必要な道具・環境
28. **successStories**: 成功事例・体験談

## TEN 6項目評価基準（習慣特化版）

### 評価算出式
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### 各項目評価基準

#### **即効性 (20%) - 効果実感の早さ**
- 100点: 1日以内に効果を実感・即座に変化
- 90点: 1週間以内に効果を実感・習慣化開始
- 80点: 2-3週間で効果を実感・継続習慣化
- 70点: 1ヶ月以内に効果を実感・安定化
- 60点: 2-3ヶ月で効果を実感・長期継続

#### **簡単さ (20%) - 導入・継続の容易さ**
- 100点: 誰でも今すぐ開始可能・継続容易
- 90点: 数分で開始・最小限の準備で継続可能
- 80点: 短時間で習得・一般的な継続努力
- 70点: 少しの準備・標準的な継続難易度
- 60点: ある程度の学習・継続に意志力必要

#### **人気度 (15%) - 認知度・実践者数**
- 100点: 誰もが知る・広く実践される習慣
- 90点: 高い認知度・多くの成功事例
- 80点: 一定の認知・専門分野で推奨
- 70点: 成長中・効果的と評価される
- 60点: ニッチ・特定グループで実践

#### **コスパ (15%) - 費用対効果**
- 100点: 完全無料・高い効果・優れたコスパ
- 90点: 最小コスト・十分な効果・良好なコスパ
- 80点: 適正コスト・相応の効果・標準的コスパ
- 70点: やや高コスト・相応の価値
- 60点: 高コスト・限定的な対象者

#### **機能専門性 (20%) - 専門性・差別化要素**
- 100点: 他にない独自効果・圧倒的な専門性
- 90点: 高度な専門効果・明確な差別化
- 80点: 優れた専門効果・特定分野で強み
- 70点: 一般的専門効果・標準以上の成果
- 60点: 基本的効果・特別な差別化なし

#### **生産性UP度 (10%) - 実際の生産性・人生向上効果**
- 100点: 劇的な生産性向上・人生の質大幅改善
- 90点: 明確な生産性向上・生活品質改善
- 80点: 一定の生産性向上・習慣化による改善
- 70点: 軽度の生産性向上・心理的満足
- 60点: 限定的な効果・主観的改善

## 品質基準
- データ精度：査読済み研究・専門書データ優先
- 実践効果：実際の習慣化成功事例・継続データ重視  
- 日本適応：日本の文化・ライフスタイルへの適応評価
- 科学的根拠：具体的研究データ・統計数値必須
- 継続性：長期継続・習慣化の実現性を検証
```
