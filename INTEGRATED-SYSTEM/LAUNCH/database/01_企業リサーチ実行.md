## 🔍 リサーチ項目（必須15項目）

### **基本情報**
1. **company_name**: 会社名（正式名称）
2. **industry**: 業界分類
3. **headquarters**: 本社所在地
4. **established**: 設立年
5. **employees**: 従業員数

### **財務・規模情報**
6. **revenue**: 売上高（億円）
7. **market_cap**: 時価総額（億円、上場企業のみ）
8. **listing_status**: 上場区分（東証プライム/スタンダード/グロース/非上場）

### **採用・働き方情報**
9. **starting_salary**: 初任給（学部卒/院卒）
10. **average_salary**: 平均年収（万円）
11. **annual_holidays**: 年間休日数
12. **overtime_hours**: 月平均残業時間
13. **paid_leave_rate**: 有給取得率（%）

### **多様性・働きやすさ**
14. **female_manager_ratio**: 女性管理職比率（%）
15. **male_parental_leave_rate**: 男性育児休業取得率（%）

## 📝 ChatGPTプロンプトテンプレート

```
【企業リサーチ指示】

業界：[業界名]

以下の形式でこの業界の主要企業をリサーチしてください：

## リサーチ要件
1. 対象企業数：15-30社程度（業界規模に応じて）
2. 優先順位：売上高・知名度・新卒採用実績の高い順
3. データ年次：2024-2025年最新データ
4. 情報源：公式IR情報・有価証券報告書・採用サイト優先

## 出力形式
以下のJSON構造で出力してください：

```json
{
  "industry": "[業界名]",
  "last_updated": "2025-09-03",
  "companies": [
    {
      "company_name": "会社名",
      "industry": "業界分類",
      "headquarters": "本社所在地",
      "established": 設立年,
      "employees": 従業員数,
      "revenue": 売上高(億円),
      "market_cap": 時価総額(億円),
      "listing_status": "上場区分",
      "starting_salary": {
        "bachelor": 学部卒初任給(万円),
        "master": 院卒初任給(万円)
      },
      "average_salary": 平均年収(万円),
      "annual_holidays": 年間休日数,
      "overtime_hours": 月平均残業時間,
      "paid_leave_rate": 有給取得率(%),
      "female_manager_ratio": 女性管理職比率(%),
      "male_parental_leave_rate": 男性育休取得率(%)
    }
  ]
}
```

## 品質基準
- データ精度：公式情報のみ使用
- 推測値：明確に「推定」と記載
- 不明項目：null値で対応
- 情報年次：2024年以降の最新データ優先
```
