## 🔧 **個別バッチ実行手順**

### **STEP 1: バッチ担当確定**
```bash
# 例：Batch 3（Q031-Q045）を担当する場合
cd app/data/qualificationDatabase/research-batches/
touch batch-03-Q031-Q045.json
```

### **STEP 2: 必須ドキュメント読了**
以下のドキュメントを順番に読んでください：
1. **完全ガイド**: `資格ランキングリール作成_完全ガイド_2025-09-05.md` - プロジェクト全体理解
2. **ガイドライン**: `INTEGRATED-SYSTEM/SPECS/database-specs/Q02_資格データベース構築ガイドライン.md` - 技術仕様・データ入力ルール
3. **実行計画**: `INTEGRATED-SYSTEM/SPECS/database-specs/Q01_資格リサーチ実行計画.md` - 詳細な作業手順

### **STEP 3: 対象資格リスト確認**
**121資格の完全リストは以下に記載**:
- `app/data/qualificationDatabase/qualificationMasterData.json` - 基本リスト（121資格）

### **STEP 4: リサーチ実行**
- ガイドラインに従って17項目を調査
- 5軸評価スコア算出（定量的基準に従って）
- 指定JSONフォーマットでデータ作成

### **STEP 5: 品質チェック**
```bash
# バリデーション実行
node scripts/validateData.js batch-03-Q031-Q045.json
```

## 💾 **統合処理（全バッチ完了後）**

### **Phase 2: データ統合**
```bash

# → qualificationMasterData.json生成
```

### **Phase 3: スコア計算・検証**
```bash
node scripts/calculateScores.js

```

### **Phase 4: ランキング生成**

# 生成されるランキング：
# - 講習のみで取得可能TOP15
# - 取得コスト最安TOP20
# - 収入インパクト最高TOP10
