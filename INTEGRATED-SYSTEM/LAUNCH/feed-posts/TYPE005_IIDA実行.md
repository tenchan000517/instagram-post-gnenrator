## 🛠️ 生成プロセス（6ステップ就活指導型）

### **STEP1: 就活段階の特定**
- [ ] 就活生の現在フェーズを把握（自己分析・企業研究・選考対策等）
- [ ] 該当するロードマップ章を選定
- [ ] 就活生の具体的な悩み・課題を特定
- [ ] 実践的解決策の方向性を設定

### **STEP2: ロードマップ章の活用**
- [ ] 該当章の核心内容を抽出
- [ ] 就活生が即座に使える部分を優先選択
- [ ] 10章間の連携・相乗効果を考慮
- [ ] 段階的成長プロセスに組み込み

### **STEP3: 企業データベース連携**
- [ ] 企業DB（152社・23業界）から関連情報を抽出
- [ ] 業界別・企業別の具体的データを活用
- [ ] 就活生の志望業界に合わせてカスタマイズ
- [ ] 実際の選考情報・内定者データを参照

### **STEP4: 実践的価値の最大化**
- [ ] 抽象論ではなく具体的な行動指針を提示
- [ ] 就活で本当に使える実用的情報に集中
- [ ] チェックリスト・テンプレート形式で提供
- [ ] 確実な成長を約束できる内容に調整

### **STEP5: IIDAキャラクター調整**
- [ ] 真面目で責任感ある指導トーンに統一
- [ ] 「〜しましょう」「〜してください」の指導口調
- [ ] 実践価値を強調する表現を挿入
- [ ] 就活生の成長を心から応援する姿勢を表現

### **STEP6: 就活実用性確認**
- [ ] 実際の就活シーンで使える内容か確認
- [ ] 他の就活生との差別化につながるか検証
- [ ] 内定獲得・選考通過に直結するか評価
- [ ] 長期的なキャリア形成に寄与するか確認

## 🔧 トラブルシューティング

### **実践性不足の場合**
**症状**: 抽象的で使えない内容
**対処**: STEP4で具体的行動指針・チェックリスト形式に変更

### **ロードマップ活用不足の場合**
**症状**: 10章の内容が十分活用されていない
**対処**: STEP2でロードマップ章を再確認・深掘り

### **差別化要素不足の場合**
**症状**: 一般的な就活アドバイスに留まる
**対処**: STEP3で企業DB連携・独自データを追加
