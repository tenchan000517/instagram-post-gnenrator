## 🎯 **システム全体構成**

### **📋 システムの目的**
Instagram投稿（フィード・リール）の自動生成システム。5キャラクター×8データベースで、女性・男性・就活生の全ターゲットに対応した高品質コンテンツを効率的に生成。

### **⚡ 達成目標**
- **効率化**: 1コンテンツ20分以内での生成
- **品質**: 手動修正不要レベル（95%以上の品質）
- **多様性**: 5キャラクター×8データベースの組み合わせ
- **自動化**: 起動術式実行のみで完成品質達成

## 🛠️ **システム動作フロー**

### **フィード投稿生成フロー**
```
1. キャラクター選択（TYPE001-005）
2. 対象ナレッジID指定
3. 起動術式実行
4. テンプレート自動選択・適用
5. キャラクター特化調整
6. 最終品質チェック・出力
```

### **リール投稿生成フロー**  
```
1. キャラクター選択（MISAKI等）
2. コンテンツ番号指定
3. 統合マスター起動術式実行
4. 10ページ構成自動生成
5. ナレーション最適化
6. キャプション生成・完成
```

### **データベースリサーチフロー**
```
1. 対象データベース選択（01-09）
2. リサーチ起動術式実行
3. 業界・分野別データ収集
4. 評価基準適用・分析
5. ランキング生成・統合
6. データベース更新・保存
```
