## 🔍 除外理由分析

### **主要除外理由トップ5**

**1. 年齢幅の過度な広さ（全体の35%）**
- "20代" "30代" = 10年幅（基準: 3歳以内）
- "20代前半・後半" = 5年幅（基準: 3歳以内）
- "中堅・若手・ベテラン" = 年数幅不明

**2. 職業の抽象化（全体の28%）**
- "会社員・サラリーマン・OL" = 業界・職種不明
- "ビジネスパーソン・社会人" = 職業内容不明
- "働く人・人材" = 職業特定不可

**3. 志向性による分類（全体の22%）**
- "効率化志向・成長志向" = 内面的特性（実在性不明）
- "向上心・キャリア志向" = 心理的傾向（測定不可）
- "スキルアップ志向" = 目標の抽象化

**4. 状況の曖昧記述（全体の10%）**
- "転職を考える・キャリアアップ希望" = 具体的行動・時期不明
- "業務効率化を図りたい" = 業務内容・課題不明
- "成果を出したい" = 成果内容・指標不明

**5. 組織・環境の非特定（全体の5%）**
- "企業・組織・チーム" = 具体的所属先不明
- "職場・部署" = 業界・規模・役割不明
- "業界・分野" = 特定領域不明
