## 🔄 実行フロー概要

### データベース作成フロー
```
1. 企業リサーチ（ChatGPT）→ JSON形式で保存
2. 業界別JSON → 統合DB（companyMasterData.json）作成
3. ランキング生成（90パターン：JS/FC/MP各30）
4. TENデータベース構築（AIツール評価・TENスコア算出）
```

### フィード投稿作成フロー
```
データベースベース:
- TEN: AIツールDB → ランキング → Type003投稿（K6xx/K9xx）
- KIKUYO: 企業DB → ランキング → Type003投稿（K8xx）

コンテンツベース:
- MISAKI: 女性向け100個 → Type003投稿
- KING: 社会人向け100個 → Type003投稿  
- IIDA: ロードマップ10章 → Type003投稿
```

### リール投稿作成フロー
```
全キャラクター共通:
1. 元コンテンツ選択
2. 10ページ構成作成
3. ナレーション生成
4. キャプション・ハッシュタグ生成
5. character_content_xxx.md として保存
```
