### 🔧 スコア修正方法
```javascript

{
  "toolName": "Fish Audio",
  "tenEvaluation": {
    "immediacy": 85,        // 即効性
    "simplicity": 80,       // 簡単さ
    "popularity": 80,       // 人気度（修正例：50→80）
    "costPerformance": 90,  // コスパ
    "specialization": 90,   // 機能専門性
    "productivityGain": 85, // 生産性UP度（修正例：75→85）
    "totalScore": 510       // 自動再計算：85+80+80+90+90+85=510
  }
}
```
