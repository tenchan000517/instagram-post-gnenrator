## ページ4: 実践的ハウツー
ブラック企業を見極める就活生向けチェックリスト

### チェックリスト形式
□ ホワイト認定をチェック
  健康経営優良法人・くるみん認定の有無を確認

□ 口コミサイトで実態調査
  OpenWork評価3.5以上、残業時間の実態をチェック

□ 説明会で有給取得率を質問
  「有給取得率と離職率データを教えてください」と聞く

□ 労基違反歴を調べる
  厚生労働省の企業名公表リストで過去の違反歴を確認
