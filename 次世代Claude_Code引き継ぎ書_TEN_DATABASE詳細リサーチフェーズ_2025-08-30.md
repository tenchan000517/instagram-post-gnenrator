## 🎯 V3統一TEN評価軸（必須理解）

### **統一済み評価軸**
```
TENスコア = 即効性(20%) + 簡単さ(20%) + 人気度(15%) + コスパ(15%) + 機能専門性(20%) + 生産性UP度(10%)
```

### **重要変更点**
- ❌ 旧「めんどくさがり対応(10%)」
- ✅ 新「生産性UP度(10%)」
- ✅ 新「機能専門性(20%)」追加

### **評価基準**
- **生産性UP度**: 実際の生産性向上効果（100点=圧倒的効果、20点=効果なし）
- **機能専門性**: 専門的価値・差別化要素（100点=業界標準レベル、20点=汎用的）
